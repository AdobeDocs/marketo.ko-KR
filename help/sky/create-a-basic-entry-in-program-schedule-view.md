# 프로그램 예약 보기에서 기본 항목 만들기

<br> 

프로그램 예약 보기에서 직접 새 기본 항목을 만들 수 있습니다.

1. 트리에서 프로그램을 선택한 다음 [!UICONTROL **개요 아래의 예약**] 탭으로 [!UICONTROL **이동합니다**].

   ![이미지 원](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-1.png)

1. 항목을 추가할 날짜 위로 마우스를 가져간 다음 [!UICONTROL **추가를 클릭합니다**].

   ![이미지 2](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-2.png)

   >[!NOTE]
   >
   >유사한 단계를 수행하여 일, 주 또는 안건 보기에서 기본 항목을 만들 수도 있습니다.

1. 새 기본 항목의 이름을 입력한 다음 필요한 경우 [!UICONTROL **유형**] 을 [!UICONTROL Basic Entry] 변경합니다. 시작 및 종료 날짜 [!UICONTROL **를**]&#x200B;수정하고 항목 [!UICONTROL **을 만든 후 달력에 유지하려면**] 안건 보기에서 열기를 선택 취소할 수도 있습니다. 저장을 [!UICONTROL **클릭합니다**].

   ![이미지 3](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-3.png)

1. 프로그램 일정 보기에서 항목을 찾을 수 있습니다. 마우스를 가져가면 자세한 내용이 표시됩니다..

   ![이미지 4](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-4.png)

...을 클릭하여 편집기 창을 엽니다.

![이미지 5](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-5.png)
