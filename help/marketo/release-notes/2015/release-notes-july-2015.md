## 리치 텍스트 편집기 업데이트 {#rich-text-editor-update}

간소화된 텍스트 서식, 이미지 편집, 링크 삽입, HTML 편집 등 최신 모양과 느낌으로 텍스트 편집기를 업데이트했습니다. 이제 HTML 편집기에 최소한의 유효성 검사가 적용되므로 코드 편집의 제한을 줄일 수 있습니다.
`<iframe width="420" height="315" src="https://www.youtube.com/embed/LmmBN6IQrII" frameborder="0" allowfullscreen></iframe>` 이 업데이트는 7월 릴리스 후 며칠 이내에 자동으로 실행됩니다. 그런 다음 편집기의 새 버전과 이전 버전 간을 관리 > 이메일 > **편집기 설정 편집에서 전환할 수 있습니다**. 새로운 버전의 텍스트 편집기에 문제가 있는 경우 알고 싶습니다. 간단한 설명을 보내주십시오 [`[email protected]`](http://docs.marketo.com/cdn-cgi/l/email-protection#d0bab3bfbfa0b5a2bdb1be90bdb1a2bbb5a4bffeb3bfbd).

## 이메일 전달 기능 단일 사인온 {#email-deliverability-single-sign-on}

이메일 전달 기능 타일을 클릭할 때 로그인 자격 증명을 더 이상 제공할 필요가 없습니다.

## 캠페인 우선 순위 지정 {#campaign-prioritization}

여러 개의 개인화된 RTP 캠페인을 설정했으며, 이러한 캠페인 중 일부가 다른 캠페인과 겹칠 수 있다는 사실을 인지한 적이 있습니까? 먼저 진행하여 캠페인의 RTP가 다른 캠페인보다 우선할 사항을 설정합니다.

![](assets/image2015-7-9-20-3a20-3a58.png)

## 회사 API {#company-api}

**REST API를 통한 회사 객체 액세스**:이제 REST API에서 Marketing Company(a.k.a. Account) 개체에 대한 액세스를 제공합니다. 즉, Marketing To에서 만든 회사 개체를 읽고 업데이트 및 삭제할 수 있으며, 업데이트된 리드 API를 사용하여 리드를 해당 회사와 연결할 수 있습니다.

회사 API에 대한 참조 가이드에서 [자세히](http://developers.marketo.com/documentation/company-api/) 알아보십시오.

## 이메일 전달 기능에 액세스 {#access-email-deliverability}

**액세스 이메일 제공 도구:**이 새로운 권한을 통해 관리자는 사용자에게 이메일 전달 가능 도구에 대한 액세스 권한을 부여할 수 있습니다.
