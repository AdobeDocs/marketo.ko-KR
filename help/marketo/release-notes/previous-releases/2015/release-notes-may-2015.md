# 릴리스 노트: 2015년 5월 {#release-notes-may}

다음 기능은 2015년 5월 릴리스에 포함되어 있습니다. 기능을 사용할 수 있는지 Marketo Edition을 확인하십시오. 릴리스 후 다시 돌아와 각 기능에 대한 세부 문서에 대한 링크를 확인하십시오!

## 응답형 랜딩 페이지

[응답형 랜딩 페이지](/help/marketo/product-docs/demand-generation/landing-pages/guided-landing-pages/create-a-guided-landing-page.md)

새로운 랜딩 페이지 편집 모드 및 템플릿 구문을 출시했습니다. &quot;자유 형식&quot; 랜딩 페이지 편집기와 달리, 새로운 &quot;안내&quot; 랜딩 페이지 편집기는 완전한 응답형 랜딩 페이지를 위한 구조화된 편집 환경을 제공합니다.

![](assets/image2015-5-15-13-3a33-3a11.png)

## 이메일 프로그램 중단

[이메일 프로그램 중단](/help/marketo/product-docs/email-marketing/email-programs/email-program-actions/abort-email-program.md)

이메일 프로그램이 출시되기 전에 전송을 입력했습니까? 새 이메일 프로그램 중단 버튼을 사용하여 브레이크를 끌어당깁니다. 이것은 비행 중 이메일 프로그램을 그들의 궤도에 바로 멈출 것입니다.

## 전자 메일 게재 기능  {#email-deliverability}

이제 Marketo은 추가된 도메인에서 매주 자동화된 SPF 및 DKIM 검사를 실행합니다. 알림을 확인하여 계속 이 작업을 수행할 수 있습니다.

## 이메일 템플릿 동작 변경 {#email-template-behavior-change}

이 릴리스부터 유효한 HTML 주석은 이제 새 이메일을 만들 때 허용되지 않으며 제거되지는 않습니다.

## RTP: 세그먼트 편집기 끌어다 놓기 {#rtp-drag-and-drop-segment-editor}

RTP: [세그먼트 편집기 끌어다 놓기](/help/marketo/product-docs/web-personalization/using-web-segments/web-segments.md)

기준을 세그먼트 빌더에 끌어다 놓고, 값을 정의하면 실시간 세그먼트를 만드는 데 도움이 됩니다.

## RTP: Predictive Content Recommendations {#rtp-predictive-content-recommendations}

[Predictive Content Recommendations](/help/marketo/product-docs/predictive-content/enabling-predictive-content/enable-predictive-content-for-web-rich-media.md)

RTP의 기계 학습 및 예측 분석 알고리즘을 사용하여 적합한 잠재 고객에게 적합한 컨텐츠를 추천합니다. 이미지 및 텍스트 설명을 통해 시각적으로 컨텐츠 자산을 향상시키고 두 개 이상의 컨텐츠 자산을 추천합니다.
