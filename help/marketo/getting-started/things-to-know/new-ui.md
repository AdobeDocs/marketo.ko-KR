# Marketo Engage 새 UI {#new-ui}

새로운 Marketo Engage UI 베타에 참여해 주셔서 감사합니다. 이 업데이트는 Marketo Engage의 스타일을 현대화하고 기능을 변경하지 않고 응답성을 개선합니다. Marketo Engage에 있는 대부분의 페이지의 오른쪽 상단 모서리에 표시되는 드롭다운을 사용하여 새 UI에 액세스합니다.

## 시작하기 전에 {#before-starting}

새 UI에 액세스하려면 먼저 다음을 수행해야 합니다.

* 하나 이상의 Marketo Engage 사용자 역할에 대해 _새 UI에 액세스_ 권한이 제공되었습니다.

* 메시지가 표시되면 Beta 테스트 열기 용어를 수락했습니다.

## 새 UI 권한 {#new-ui-permission}

관리자는 하나 이상의 사용자 역할에 _새 UI 액세스_ 권한을 부여할 수 있습니다.

1. **관리자** 영역에서 **사용자 및 역할**&#x200B;을 선택합니다.

   ![](assets/new-ui-1.png)

1. **역할** 탭을 클릭합니다. 원하는 역할을 선택하고 **역할 편집**&#x200B;을 클릭합니다.

   ![](assets/new-ui-2.png)

>[!NOTE]
>
>새 역할을 만들 수도 있습니다.

1. **새 테마 액세스** 확인란을 선택하고 **저장**&#x200B;을 클릭합니다.

   ![](assets/new-ui-3.png)

## 신규 및 클래식 UI {#new-and-classic}

새 UI로 전환하려면 오른쪽 상단의 UI 드롭다운을 클릭하고 **새로 만들기(Beta)**&#x200B;를 선택합니다.

![](assets/new-ui-4.png)

어떤 이유로든 다시 전환해야 하는 경우 UI 드롭다운을 다시 클릭하고 **클래식**&#x200B;을 선택합니다.

## 피드백 제출 중 {#feedback}

귀하의 피드백을 환영합니다. 새 UI를 탐색하는 동안 기능에 액세스하거나 사용하는 데 문제가 있거나 제안 또는 문제가 있는 경우 오른쪽 상단의 UI Beta 피드백 버튼을 클릭합니다.

![](assets/new-ui-5.png)
