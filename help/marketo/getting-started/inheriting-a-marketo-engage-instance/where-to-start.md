# 시작 위치 {#where-to-start}

새 작업을 시작하거나 다른 관리자로부터 기존 인스턴스를 인수하고 있으며 어디서부터 시작해야 할지 잘 모르겠습니까? 얼마 동안 실행 중인 라이브 인스턴스를 인수하면 약간 위협적으로 보일 수 있지만, 사용자가 즉시 속도를 높일 수 있도록 몇 가지 리소스를 구성했습니다.

## Adobe Marketo Engage 커뮤니티에서 설정 {#get-set-up}

Marketo Engage 커뮤니티에서 제대로 설정되었는지 확인하십시오.

* 필수 필드(회사, 솔루션 전문 분야, 국가 등)를 입력하여 [**프로필을 설정**](https://experienceleaguecommunities.adobe.com/adobe-marketo-engage-26){target="_blank"}

* [**Adobe Experience League 커뮤니티 지침**](https://experienceleaguecommunities.adobe.com/knowledge-base){target="_blank"}을 읽어 보십시오.

* [**지원 사례를 제출**](https://experienceleague.adobe.com/en/support#home){target="_blank"}하고 [지원 관리자](https://experienceleague.adobe.com/en/docs/support-resources/adobe-support-tools-guide/admin-roles#administrative-hierarchy){target="_blank"}(으)로 설정되어 있는지 확인하는 방법을 알아보세요.

## Marketo Engage 커뮤니티 프로필 병합 {#merge-your-community-profiles}

이전 회사에 Marketo Engage 커뮤니티 프로필이 이미 있는 경우 다음 단계에 따라 해당 기록을 새 프로필로 가져오십시오.

### 1단계: 새 커뮤니티 프로필 만들기 {#create-a-new-profile}

1. 현재 Marketo Engage 구독에 액세스할 수 있는 회사 이메일 주소 또는 개인 이메일 주소를 사용하여 사용자 이름 및 커뮤니티 프로필을 만듭니다.

1. 회사, 솔루션 전문 분야, 국가 등과 같은 필드를 채웁니다.

### 2단계: 병합 요청 제출 {#submit-merge-request}

새 계정 정보로 프로필을 만든 후 <a href="mailto:&#101;&#120;&#108;&#99;&#111;&#109;&#109;&#117;&#110;&#105;&#116;&#121;&#64;&#97;&#100;&#111;&#98;&#101;&#46;&#99;&#111;&#109;">에 요청을 제출하세요. 이전 커뮤니티 프로필의 데이터를 새 프로필과 병합하는 exlcommunity@adobe.com</a>. 요청에서 다음 세부 정보를 지정합니다.

* 이전 커뮤니티 프로필과 연결된 이전 이메일 주소
* 현재 Marketo Engage 구독과 연결된 새 회사 이메일 주소 또는 새 프로필을 만드는 데 사용하는 경우 개인 이메일(해당하는 경우)
* 새 사용자 이름/커뮤니티 프로필 URL

### 3단계: 병합 완료 및 다시 로그인 {#complete-merge}

Adobe 커뮤니티 팀은 영업일 기준으로 4~6일 내에 커뮤니티 프로필을 병합합니다. 완료되면 로그인 시 새 커뮤니티 프로필에 참여 기록이 표시됩니다.

참여 기록이 표시되지 않으면 다음을 시도해 보십시오.

* 계정에서 로그아웃하고 다시 로그인하여 확인
* 브라우저 캐시 지우기
* 시크릿/개인 브라우저를 사용하여 로그인
* 계정 ID가 새로 고쳐질 때까지 약 2시간 기다린 다음 다시 확인

## 도움말 받는 방법 {#ways-to-get-help}

도움이 더 필요하십니까? 아래 링크를 통해 당사 또는 동료 Marketo Engage 사용자와 연결합니다.

* 제대로 작동하지 않는 문제에 대한 기술 지원이 필요하면 **[Marketo Engage 고객 지원 센터](https://experienceleague.adobe.com/en/support){target="_blank"}**&#x200B;에 문의하십시오.

* **[Adobe Marketo Engage 커뮤니티](https://experienceleaguecommunities.adobe.com/adobe-marketo-engage-26){target="_blank"}**&#x200B;에서 동료 Marketo Engage 사용자와 연결하고 학습합니다.

* Marketo Engage 인스턴스를 최대한 활용하는 데 도움이 필요하면 **[Adobe Professional Services](https://business.adobe.com/products/marketo/services-support.html){target="_blank"}**&#x200B;에 문의하세요.

* Marketo Engage의 다양한 측면에서 **[큐레이션된 튜토리얼 비디오](https://experienceleague.adobe.com/docs/marketo-learn/tutorials/overview.html){target="_blank"}**&#x200B;를 시청하십시오.
