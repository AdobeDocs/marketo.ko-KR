# 도움말! 기존 Marketo 인스턴스를 상속했습니다.

**기존 Marketo 인스턴스를 방금 상속했으며 어디서부터 시작해야 할지 모르는 새 사용자입니까?**

그렇다면 이 가이드는 당신을 위한 것입니다. 필요한 것을 얻기 위해 올바른 방향으로 안내하여 Marketo을 처음 사용하는 날을 최대한 활용하는 데 도움이 됩니다.
