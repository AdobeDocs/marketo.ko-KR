# 상속된 Marketo Engage 인스턴스 Jumpstart {#jumpstart-an-inherited-marketo-engage-instance}

새 작업을 시작하거나 다른 관리자의 기존 인스턴스를 인수하시겠습니까? 잠시 동안 실행되는 라이브 인스턴스를 인수하는 것은 다소 어려울 수 있지만 이 안내서가 도움이 될 것입니다.

Marketo Engage 인스턴스 감사 관리 체크리스트를 사용하여 인스턴스를 검토 및 테스트하고, 오류를 해결하며, 문제가 되기 전에 빨간색 플래그를 조사하십시오.
