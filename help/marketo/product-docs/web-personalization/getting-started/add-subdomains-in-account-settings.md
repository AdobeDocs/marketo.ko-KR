# 계정 설정에서 하위 도메인 추가 {#add-subdomains-in-account-settings}

계정 설정에서 기본 도메인에 하위 도메인을 추가하는 방법을 설명합니다. 이렇게 하면 주 도메인의 특정 RTP Javascript와 관련된 하위 도메인을 관리할 수 있습니다. 추가된 하위 도메인에 Javascript 태그를 배포하는 것이 좋습니다.

1. 웹 개인화에서 **계정 설정으로 이동합니다**.

   ![](assets/image2014-12-1-23-3-12.png)

1. 도메인 구성 페이지에서 계정과 연관된 모든 기본 도메인 목록이 표시됩니다. 각 섹션에는 기본 도메인이 먼저 표시되고(아래 강조 표시) 하위 도메인이 표시됩니다. 하위 **도메인 추가를 클릭합니다**.

   ![](assets/highlightprimary2.png)

1. 추가를 **클릭합니다**.

   ![](assets/add.png)

1. 하위 도메인 URL을 입력합니다. 추가 **또는** 삭제 **를 클릭하여 하위 도메인 목록을** 관리하고 **완료되면** 확인을클릭합니다.

   ![](assets/newsubdomain.png)

1. 새로 추가된 하위 도메인이 나열됩니다.

   ![](assets/finalnew.png)

   >[!NOTE]
   >
   >계정에 *기본* 도메인을 추가하려면 Marketing [To 지원에 문의하십시오](http://docs.marketo.com/cdn-cgi/l/email-protection#5e2d2b2e2e312c2a1e333f2c353b2a31703d3133).
