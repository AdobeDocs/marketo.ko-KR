# 추적되지 않도록 웹 개인화 설정 {#setting-web-personalization-to-do-not-track}

웹 방문자는 &quot;DNT(추적 안 함)&quot;를 선택하여 모든 웹 사이트의 추적을 방지하도록 브라우저를 설정할 수 있습니다. 이렇게 하면 해당 특정 브라우저 및 장치에 대한 추적이 방지됩니다.

웹 개인화 및 예측 컨텐츠에서 마케터는 토글을 설정하여 브라우저의 DNT(Do Not Track) 설정을 지원할지 아니면 무시할지를 지정할 수 있습니다. 계정 토글이 기본값으로 설정되지 않은 것으로 설정되어 있습니다. 즉, DNT가 애플리케이션에서 적용되지 않습니다.

## 전환 활성화 또는 비활성화 {#enable-or-disable-the-toggle}

1. 이동 **계정 설정**.

   ![](assets/image2014-12-1-23-3a3-3a12.png)

1. 도메인 및 도메인 구성에서 **설정** DNT 인증 토글을 사용하도록 설정합니다.

   ![](assets/two-1.png)

   토글이 켜짐으로 설정되면 웹 개인화는 브라우저의 DNT(Do Not Track) 설정을 존중하고 지원하며, 웹 활동을 추적하거나 웹 사이트에서 캠페인이나 콘텐츠 권장 사항을 실행하지 않습니다.

   >[!NOTE]
   >
   >토글을 켜기로 설정하면 특정 영역의 Marketo 값 및 기능에 영향을 줄 수 있습니다.

1. DNT 인증 전환을 비활성화하고 브라우저의 DNT(Do No Track) 설정을 무시하려면 을 선택합니다. **해제** DNT에서 사용할 수 있습니다.
