# 웹 개인화를 추적 안 함으로 설정 {#setting-web-personalization-to-do-not-track}

웹 방문자는 &quot;추적 안 함&quot;(DNT)을 선택하여 모든 웹 사이트에 의한 추적을 방지하도록 브라우저를 설정할 수 있습니다. 이렇게 하면 해당 특정 브라우저 및 장치에 대한 추적이 방지됩니다.

웹 개인화 및 예측 콘텐츠에서 마케터는 브라우저의 DNT(추적 안 함) 설정을 지원할지 또는 무시할지 여부를 표시하도록 토글을 설정할 수 있습니다. 계정에 대한 토글이 기본값으로 꺼져 있습니다. 즉, 애플리케이션에서 DNT를 사용하지 않습니다.

## 토글 활성화 또는 비활성화 {#enable-or-disable-the-toggle}

1. 다음으로 이동 **계정 설정**.

   ![](assets/image2014-12-1-23-3a3-3a12.png)

1. 도메인 및 도메인 구성에서 다음을 선택합니다. **날짜** Honor DNT 토글을 활성화합니다.

   ![](assets/two-1.png)

   토글이 켜짐으로 설정되면 웹 개인화는 브라우저의 DNT(추적 안 함) 설정을 적용 및 지원하며 웹 활동을 추적하거나 웹 사이트에서 캠페인 또는 콘텐츠 권장 사항을 실행하지 않습니다.

   >[!NOTE]
   >
   >토글을 켜짐으로 설정하면 특정 영역의 Marketo 값과 기능에 영향을 줄 수 있습니다.

1. DNT 승인 토글을 비활성화하고 브라우저의 DNT(추적 안 함) 설정을 무시하려면 다음을 선택합니다. **끔** 명예 DNT에 속해 있습니다.
