# 웹 캠페인이 표시되는 방식 설정 {#set-how-your-web-campaign-displays}

웹 캠페인이 표시되는 시기/방법에 대해서는 몇 가지 방법을 사용할 수 있습니다.

## 페이지 로드 {#page-load}

기본적으로 이 선택 사항은 페이지 로드 시 캠페인을 간단히 표시합니다.

![](assets/pl1.png)

## 지연 {#delay}

웹 캠페인이 웹 사이트에서 반응할 시간 지연(총 초)을 지정합니다.

1. 을(를) 클릭합니다. **표시 켜기** 드롭다운 및 선택 **지연**.

   ![](assets/d1.png)

1. 을(를) 클릭합니다. **대기** 드롭다운을 클릭하고 원하는 시간을 선택합니다.

   ![](assets/d2.png)

## 스크롤 {#scroll}

1. 을(를) 클릭합니다. **표시 켜기** 드롭다운 및 선택 **스크롤**.

   ![](assets/s1.png)

1. 을(를) 클릭합니다. **When** 드롭다운을 클릭하고 웹 캠페인을 표시할 시기를 선택합니다.

   ![](assets/s2.png)

  </tr>

## 종료 의도 {#exit-intent}

종료 의도는 브라우저의 마우스 커서 종료 시 웹 캠페인을 표시합니다.

1. 을(를) 클릭합니다. **표시 켜기** 드롭다운 및 선택 **종료 의도**.

   ![](assets/ei1.png)

1. 종료 의도가 모바일과 호환되지 않음을 알리는 메시지가 나타납니다.

   ![](assets/ei2.png)
