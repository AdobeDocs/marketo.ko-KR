# 웹 캠페인 데이터 내보내기 {#export-web-campaign-data}

웹 캠페인 데이터를 내보내려면 다음 간단한 단계를 따르십시오.

1. 웹 **캠페인으로 이동합니다**.

   ![](assets/one-2.png)

1. 페이지 오른쪽 상단에서 CSV 내보내기 아이콘을 클릭합니다.

   ![](assets/two-2.png)

1. 파일을 열거나 저장합니다.

   ![](assets/three-2.png)

1. 유용한 통계를 보려면 파일을 확인하십시오.

   ![](assets/four-1.png)
