# 웹 캠페인 보관 {#archive-a-web-campaign}

1. **웹 캠페인**(으)로 이동합니다.

   ![](assets/one.jpg)

1. 원하는 웹 캠페인의 상태 드롭다운을 클릭하고 **보관**&#x200B;을 선택합니다.

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >보관된 웹 캠페인은 기본 필터에 표시되지 않습니다. 이를 보려면 필터 아이콘을 클릭하고 **상태**&#x200B;에서 **보관** 확인란을 선택하고 **적용**&#x200B;을 클릭합니다.
