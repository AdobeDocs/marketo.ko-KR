# 웹 캠페인 보관 {#archive-a-web-campaign}

1. 이동 **웹 캠페인**.

   ![](assets/one.jpg)

1. 원하는 웹 캠페인의 상태 드롭다운을 클릭하고 을 선택합니다 **아카이브**.

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >보관된 웹 캠페인은 기본 필터에 표시되지 않습니다. 이를 보려면 필터 아이콘 및 **상태**&#x200B;에서 을(를) 선택합니다. **보관됨** 확인란을 선택하고 **적용**.
