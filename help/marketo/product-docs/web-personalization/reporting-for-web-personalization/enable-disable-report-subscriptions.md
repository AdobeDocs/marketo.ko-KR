# 보고서 구독 활성화/비활성화 {#enable-disable-report-subscriptions}

웹 개인화에는 이메일을 통해 전송되는 몇 가지 유용한 보고서가 있습니다. 여기에 가입하는 방법이 있습니다.

1. 웹 개인화에 로그인합니다. 로그인 이름에서 **사용자 설정**.

   ![](assets/image2014-9-17-20-3a48-3a28.png)

1. 구독할 보고서 및 빈도를 선택합니다. 클릭 **저장**.

   ![](assets/email-settings.png)

   됐습니다. 보고서의 가입을 해지하려면 선택을 취소하고 를 클릭합니다 **저장**.
