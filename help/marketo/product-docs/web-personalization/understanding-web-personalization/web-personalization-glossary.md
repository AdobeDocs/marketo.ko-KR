# 웹 개인화 용어집 {#web-personalization-glossary}

Marketo 웹 개인화의 세계 및 언어에 대한 일부 통찰력.

| 용어 | 정의 |
|---|---|
| **익명 방문자** | 양식을 작성하거나 웹 사이트에 세부 사항을 빠뜨리지 않은 웹 방문자입니다. |
| **웹 캠페인** | 특정 세그먼트와 연관된 사용자 지정된 반응입니다. 웹 개인화를 통해 웹 캠페인에는 영역 및 위젯의 대화 상자가 포함됩니다. |
| **클릭스트림** | 사이트에서의 방문자의 활동 및 URL 경로 및 각 페이지를 방문한 시간 |
| **ISP** | 인터넷 서비스 공급자 |
| **알려진 방문자** | 양식을 완료하고 웹 사이트에 세부 사항(이메일 주소)을 남겨 두거나 Marketo 이메일에 있는 링크를 클릭한 웹 방문자입니다. |
| **계정 목록** | 주요 계정/조직 이름 목록입니다. Account-Based Marketing(ABM) 목록이라고도 합니다. |
| **세그먼트** | 에 정의된 지정된 기준을 충족하는 방문자의 컬렉션입니다 [세그먼트 설정 페이지](/help/marketo/product-docs/web-personalization/using-web-segments/web-segments.md). |
| **테스트 분할** | 두 개 이상의 변형을 사용하여 테스트하여 결과의 차이를 측정합니다. 목표는 원하는 결과를 늘리거나 극대화하는 웹 페이지의 변경 사항을 식별하는 것입니다. |
| **와일드카드** | 와일드카드 문자(&#42; 문자열 앞 또는 뒤에서 사용하여 문자열의 다른 문자 또는 문자를 대체(사용) 아래 예를 참조하십시오. |
