# 지정된 계정 만들기 {#create-a-named-account}

지정된 계정을 수동으로 만들려면 다음 단계를 수행합니다.

1. 지정된 계정에서 **새로 만들기** 드롭다운을 클릭하고 **지정된 계정**&#x200B;만들기를 선택합니다.

   ![](assets/two-1.png)

1. 원하는 필드를 채우고 만들기를 **클릭합니다**.

   ![](assets/three-1.png)

   >[!TIP]
   >
   >지정된 계정을 클릭하여 대시보드를 확인합니다.
