# SEO - 페이지 추가 {#seo-add-pages}

페이지 있어요! Marketo SEO는 자동으로 사이트를 크롤링하고 추적합니다. 누락된 항목이 있을 경우 다음과 같이 SEO 앱에서 가져옵니다.

1. 로 이동합니다. **페이지** 섹션을 참조하십시오.

   ![](assets/image2014-9-18-12-3a55-3a19.png)

1. 클릭 **페이지 추가**.

   ![](assets/image2014-9-18-12-3a55-3a53.png)

1. 추가할 URL을 입력합니다. 클릭 **추가**.

   ![](assets/image2014-9-18-12-3a56-3a15.png)

   >[!TIP]
   >
   >네가 할 수 있다는 걸 알고 있었니 [페이지를 새 목록이나 기존 목록에 추가합니다](/help/marketo/product-docs/additional-apps/seo/understanding-seo/seo-managing-lists.md)? 확인해!

   페이지 데이터를 로드하는 데 시간이 걸릴 수 있습니다. 녹색 경고 메시지가 표시될 때까지 기다렸다가 페이지를 새로 고쳐 표시를 업데이트합니다.

   ![](assets/image2014-9-18-12-3a57-3a10.png)

   이제 이 페이지의 검색 성과를 추적할 수 있습니다.
