# SEO - 페이지 문제 제거/삭제 {#seo-remove-delete-a-page-issue}

모든 페이지 문제가 유용한 것은 아닙니다. 하나는 제거하는 방법입니다.

1. 로 이동 **[!UICONTROL 페이지]** 섹션.

   ![](assets/image2014-9-18-14-3a0-3a16.png)

1. 다음에서 [!UICONTROL 페이지] 섹션, 클릭 **[!UICONTROL 문제]**.

   ![](assets/image2014-9-18-14-3a0-3a30.png)

1. 제거할 페이지 문제를 마우스로 가리킵니다. 클릭 **[!UICONTROL 제거]**.

   ![](assets/image2014-9-18-14-3a0-3a38.png)

1. 클릭 **[!UICONTROL 삭제]** 이 페이지 문제를 영구적으로 제거합니다.

   >[!CAUTION]
   >
   >이 작업은 실행 취소할 수 없습니다. 문제가 제거되면 페이지를 제거하고 다시 추가하여 문제를 재구축할 수 있습니다.

   ![](assets/image2014-9-18-14-3a1-3a28.png)

이제 페이지 문제가 제거되었습니다.
