# 채팅 사용자 추가 또는 제거 {#add-or-remove-chat-users}

채팅 사용자를 추가하거나 제거하려면 아래 단계를 따르십시오.

## 채팅 사용자 추가 {#add-a-chat-user}

1. [Adobe Admin Console](https://adminconsole.adobe.com/)에 로그인합니다.

   ![](assets/add-or-remove-chat-users-1.png)

1. **동적 채팅**&#x200B;을 클릭합니다.

   ![](assets/add-or-remove-chat-users-2.png)

   >[!NOTE]
   >
   >Marketo 구독이 두 개 이상 있는 경우 다음 단계 전에 원하는 구독을 선택하십시오.

1. **사용자** 탭을 클릭합니다.

   ![](assets/add-or-remove-chat-users-3.png)

1. **사용자 추가** 단추를 클릭합니다.

   ![](assets/add-or-remove-chat-users-4.png)

1. 추가할 사용자의 이름, 사용자 그룹 또는 이메일 주소를 입력합니다. 이름과 성은 선택 사항입니다.

   ![](assets/add-or-remove-chat-users-5.png)

1. **+** 아이콘을 클릭하고 원하는 제품 프로필을 선택합니다.

   ![](assets/add-or-remove-chat-users-6.png)

1. **저장**&#x200B;을 클릭합니다.

   ![](assets/add-or-remove-chat-users-7.png)

그러면 사용자가 Marketo에 로그인하는 이메일을 받게 됩니다.

## 채팅 사용자 제거 {#remove-a-chat-user}

1. [Adobe Admin Console](https://adminconsole.adobe.com/)에 로그인합니다.

   ![](assets/add-or-remove-chat-users-8.png)

1. **동적 채팅**&#x200B;을 클릭합니다.

   ![](assets/add-or-remove-chat-users-9.png)

   >[!NOTE]
   >
   >Marketo 구독이 두 개 이상 있는 경우 다음 단계 전에 원하는 구독을 선택하십시오.)

1. **사용자** 탭을 클릭합니다.

   ![](assets/add-or-remove-chat-users-10.png)

1. 제거할 사용자를 선택합니다.

   ![](assets/add-or-remove-chat-users-11.png)

1. **사용자 제거** 단추를 클릭합니다.

   ![](assets/add-or-remove-chat-users-12.png)

1. **사용자 제거**&#x200B;를 클릭하여 확인합니다.

   ![](assets/add-or-remove-chat-users-13.png)
