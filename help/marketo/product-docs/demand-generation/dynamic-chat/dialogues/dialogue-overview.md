# 대화 상자 개요 {#dialogue-overview}

대화 상자는 개별 채팅 대화입니다. 각 대화 상자에서는 특정 채팅 대화가 표시되는 위치와 대화 내용 및 내용을 결정합니다. 또한 각 대화 상자에는 효과를 모니터링할 수 있는 자체 보고서 페이지가 있습니다.

## 대상 기준 {#audience-criteria}
