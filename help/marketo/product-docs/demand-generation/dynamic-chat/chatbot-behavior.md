# Chatbot 동작 {#chatbot-behavior}

다음은 각 사이트에서 방문자가 예상하는 Chatbot의 동작을 능가하는 다른 가능한 시나리오입니다.

<table>
  <tbody>
    <tr>
      <th>약어</th>
      <th>세부 정보</th>
    </tr>
    <tr>
      <td>D1, D2, D3 등</td>
      <td>D1이 하나의 대화 상자인 여러 대화 상자를 나타냅니다.</td>
    </tr>
    <tr>
      <td>P1, P2, P3 등</td>
      <td>P1이 가장 높은 우선 순위인 대화 상자 우선순위를 나타냅니다</td>
    </tr>
    <tr>
      <td>WP1, WP2, WP3 등</td>
      <td>WP1이 첫 번째 웹 페이지인 여러 웹 페이지를 나타냅니다.</td>
    </tr>
    <tr>
      <td>V1, V2, V3 등</td>
      <td>V1이 방문자 1인 여러 웹 페이지 방문자를 나타냅니다</td>
    </tr>
   </tbody>
</table>
