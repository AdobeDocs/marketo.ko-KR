# 회의 {#meetings}

다양한 대화 상자를 통해 웹 사이트 방문자가 예약한 모든 약속을 볼 수 있습니다. 여기에서 약속을 예약한 사람의 전자 메일 주소, 약속을 예약한 대리인, 약속 발생 예정 시간, 예약된 모임 시간이 경과되었는지 여부를 확인할 수 있습니다.
