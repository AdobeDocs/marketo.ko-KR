# 양식 편집 {#edit-a-form}

기존 양식을 변경하려면 프로세스를 시작하는 방법을 설명합니다.

1. 마케팅 **활동** 으로 **이동합니다**.

   ![](assets/login-marketing-activities.png)

1. 양식을 선택하고 양식 **편집을** **클릭합니다**.

   ![](assets/editform.png)

>[!NOTE]
>
>**딥 다이브**
>
>여기에서 [양식](http://docs.marketo.com/display/docs/forms) 편집 및 사용자 지정에 대한 자세한 내용을 살펴보십시오.
