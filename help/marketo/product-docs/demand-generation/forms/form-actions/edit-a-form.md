# {#edit-a-form} 양식 편집

기존 양식을 변경하려면 프로세스를 시작하는 방법을 설명합니다.

1. **Marketing** **활동**&#x200B;으로 이동합니다.

   ![](assets/login-marketing-activities.png)

1. 양식을 선택하고 **편집** **양식**&#x200B;을 클릭합니다.

   ![](assets/editform.png)

>[!NOTE]
>
>**자세히 알아보기**
>
>여기에서 [양식](http://docs.marketo.com/display/docs/forms)을 편집하고 사용자 지정하는 방법에 대해 자세히 알아보십시오.
