# 웹 사이트에 양식 포함 {#embed-a-form-on-your-website}

Marketo에서는 웹 사이트에 양식을 포함할 수 있습니다. 포함 코드에 대한 액세스 권한을 가져오는 방법은 다음과 같습니다.

1. 이동 **마케팅 활동**.

   ![](assets/login-marketing-activities-4.png)

1. 양식을 찾아 선택합니다.

   ![](assets/image2014-9-15-12-3a12-3a14.png)

1. 아래 **양식 작업**&#x200B;를 클릭합니다. **포함 코드**.

   >[!NOTE]
   >
   >양식에 대해 승인을 받아야 합니다. **포함 코드** 표시/사용 가능한 항목입니다.

   ![](assets/image2014-9-15-12-3a12-3a20.png)

   >[!CAUTION]
   >
   >**[양식 미리 채우기](/help/marketo/product-docs/administration/settings/edit-landing-page-settings.md)** 자체 페이지에서 양식 포함 코드를 사용할 때는 작동하지 않습니다 _또는_ Marketo 랜딩 페이지. 양식 미리 채우기는 요소 삽입 옵션을 통해 Marketo 랜딩 페이지에서 양식이 사용되는 경우에만 작동합니다.

1. 포함 코드를 선택/복사한 다음 를 클릭합니다 **닫기**.

   ![](assets/image2014-9-15-12-3a12-3a31.png)

>[!TIP]
>
>코드가 웹 사이트에 포함되면 Marketo에서 양식 변경 사항이 양식 승인 시 사이트에 푸시됩니다. 코드를 더 변경할 필요가 없습니다.

이제 웹 개발자에게 포함 코드를 제공하고 해당 코드를 사이트에 추가하도록 하면 됩니다.

>[!NOTE]
>
>개발자가 모양을 사용자 지정하거나 고급 API 기능에 액세스하려는 경우 [Forms 2.0 개발자 페이지](https://developers.marketo.com/documentation/websites/forms-2-0/).

잘했어요! 이 [lightbox 코드를 포함하는 포함 코드](/help/marketo/product-docs/demand-generation/forms/form-actions/use-a-form-in-a-lightbox.md)? 그것도 쉬워요!
