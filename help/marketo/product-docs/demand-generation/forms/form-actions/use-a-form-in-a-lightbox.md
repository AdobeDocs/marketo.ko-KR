# Lightbox에서 양식 사용 {#use-a-form-in-a-lightbox}

Lightbox는 표시하려는 경우 콘텐츠 앞에 양식을 여는 기법입니다. 방법은 다음과 같습니다.

1. 이동 **마케팅 활동**.

   ![](assets/login-marketing-activities-8.png)

1. 양식을 찾아 선택합니다.

   ![](assets/image2014-9-15-14-3a32-3a15.png)

1. 아래 **양식 작업**&#x200B;를 클릭합니다. **포함 코드**.

   >[!NOTE]
   >
   >포함 코드 항목을 표시/사용 가능하려면 양식을 승인해야 합니다.

   ![](assets/image2014-9-15-14-3a32-3a24.png)

1. 설정 **코드 유형** to **Lightbox**.

   ![](assets/image2014-9-15-14-3a32-3a31.png)

1. 코드를 선택/복사하고 를 클릭합니다 **닫기**.

   ![](assets/image2014-9-15-14-3a32-3a39.png)

코드를 웹 개발자에게 전달하고 웹 사이트에 추가하도록 합니다.

잘했어요!
