# URL 빌더 사용 {#using-the-url-builder}

URL 빌더를 사용하면 Marketo 숨김 양식 필드를 채울 수 있는 URL을 구성할 수 있습니다.

>[!PREREQUISITES]
>
>양식에서 숨김 필드를 만들고 설정을 편집하는 방법을 배웁니다. [양식 필드를 숨김으로 설정](/help/marketo/product-docs/demand-generation/forms/form-fields/set-a-form-field-as-hidden.md).

1. 랜딩 페이지를 선택하고 을(를) 클릭합니다. **랜딩 페이지 작업**&#x200B;를 마우스로 가리키기 **URL 도구**&#x200B;를 클릭한 다음 **URL 빌더**.

   ![](assets/image2014-9-18-13-3a5-3a19.png)

1. 사용할 필드를 선택하고 값을 입력한 다음 **업데이트 URL**.

   ![](assets/image2014-9-18-13-3a5-3a28.png)

   >[!NOTE]
   >
   >빌더에서 사용할 수 있는 필드가 표시되지 않으면 양식에 숨김 필드가 있는지 확인하십시오 [url 매개 변수를 수락하도록 설정](/help/marketo/product-docs/demand-generation/forms/form-fields/set-a-hidden-form-field-value.md#url-parameter).

잘했어요! 이제 URL을 복사하여 웹에서 사용할 수 있습니다.
