# URL 빌더 사용 {#using-the-url-builder}

URL 빌더는 Marketo 숨겨진 양식 필드를 채울 수 있는 URL을 구성하는 데 도움이 됩니다.

>[!PREREQUISITES]
>
>양식에서 숨겨진 필드를 만들고 해당 설정을 편집하는 방법에 대해 알아봅니다. [양식 필드를 숨김으로 설정](/help/marketo/product-docs/demand-generation/forms/form-fields/set-a-form-field-as-hidden.md).

1. 랜딩 페이지를 선택하고 다음을 클릭합니다. **랜딩 페이지 작업**, 마우스로 가리키기 **URL 도구**&#x200B;를 클릭한 다음 을 클릭합니다. **URL 빌더**.

   ![](assets/image2014-9-18-13-3a5-3a19.png)

1. 사용할 필드를 선택하고 값을 입력한 다음 을(를) 클릭합니다 **URL 업데이트**.

   ![](assets/image2014-9-18-13-3a5-3a28.png)

   >[!NOTE]
   >
   >빌더에서 사용할 수 있는 필드가 표시되지 않으면 양식에 숨겨진 필드가 있는지 및 필드가 있는지 확인하십시오 [url 매개 변수를 수락하도록 설정](/help/marketo/product-docs/demand-generation/forms/form-fields/set-a-hidden-form-field-value.md#url-parameter).

훌륭합니다! 이제 URL을 복사하여 붙여넣은 후 웹에서 사용할 수 있습니다.
