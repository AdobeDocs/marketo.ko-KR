# 랜딩 페이지 테스트 그룹 복제 {#cloning-a-landing-page-test-group}

Design Studio에서 랜딩 페이지 테스트 그룹을 찾아 복제할 수 있습니다.

1. 승인되지 않은 페이지를 마우스 오른쪽 단추로 클릭합니다. **복제**&#x200B;를 선택합니다.

   ![](assets/image2015-4-27-15-3a11-3a24.png)

1. 이 복제된 페이지에 새 **이름**&#x200B;을(를) 지정합니다. 복제한 페이지는 현재 폴더 하단에 표시됩니다.

   ![](assets/image2015-4-27-16-3a10-3a10.png)

1. 이제 **새로 만들기**&#x200B;를 클릭하고 **새 테스트 그룹**&#x200B;을 선택하여 새 테스트 그룹을 만듭니다.

   ![](assets/image2015-4-27-15-3a49-3a54.png)

1. [새 랜딩 페이지 만들기] 대화 상자에서 새 테스트 그룹 **이름**&#x200B;을 입력합니다.

   ![](assets/image2015-4-27-15-3a58-3a13.png)

1. 대화 상자의 하단에서 **사용 가능한 랜딩 페이지** 목록에서 복제된 페이지를 선택하고 오른쪽 화살표를 클릭하여 **테스트된 랜딩 페이지**&#x200B;에 포함합니다. 이 그룹에서 원하는 모든 랜딩 페이지에 대해 이 작업을 수행합니다.

   ![](assets/image2015-4-27-16-3a3-3a22.png)

1. 완료되면 **만들기**&#x200B;를 클릭합니다.

   ![](assets/image2015-4-27-16-3a7-3a50.png)

1. 테스트 그룹을 시작할 준비가 되면 트리에서 그룹을 마우스 오른쪽 단추로 클릭하고 **테스트 그룹 승인**&#x200B;을 선택합니다.

   ![](assets/image2015-4-27-16-3a19-3a10.png)

   이제 이 그룹은 어떤 랜딩 페이지가 가장 성과가 좋은지 확인할 수 있도록 방문 및 양식 완료를 적극적으로 추적합니다.
