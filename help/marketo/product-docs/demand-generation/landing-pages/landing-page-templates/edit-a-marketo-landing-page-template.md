# Marketo 랜딩 페이지 템플릿 편집 {#edit-a-marketo-landing-page-template}

Marketo 내에서 모든 랜딩 페이지 템플릿을 편집할 수 있습니다.

1. 이동 **Design Studio**.

   ![](assets/designstudio.png)

1. 확장 **랜딩 페이지** 템플릿을 표시하려면 을 선택합니다.

   ![](assets/image2015-5-21-12-3a40-3a3.png)

1. 을(를) 선택합니다 **템플릿** 편집하려는 경우 클릭 **초안 편집**.

   ![](assets/image2015-5-21-12-3a37-3a54.png)

   완료! 이제 템플릿의 CSS를 편집하고 모양과 레이아웃을 완전히 제어할 수 있습니다.

   >[!NOTE]
   >
   >랜딩 페이지 템플릿을 편집하면 해당 템플릿을 사용하여 모든 랜딩 페이지 자산의 초안을 만듭니다.
