# 랜딩 페이지 테스트 그룹 {#landing-page-test-groups}

Marketo은 테스트 그룹 내에서 테스트된 각 페이지에서 페이지 보기 및 양식 완료 수를 추적합니다. 테스트 그룹 결과를 사용하여 가장 매력적인 랜딩 페이지를 결정할 수 있습니다. 다음은 테스트 그룹을 만드는 방법입니다.

>[!PREREQUISITES]
>
>[두 개 이상의 랜딩 페이지 작성](/help/marketo/getting-started/quick-wins/landing-page-with-a-form.md)바람직하게는, 양식인 경우.

1. Design Studio에서 **새로 만들기**. 드롭다운에서 을(를) 선택합니다. **새 테스트 그룹**.

   ![](assets/image2015-8-5-13-3a32-3a50.png)

   >[!NOTE]
   >
   >테스트 그룹을 만드는 다른 방법은 다음과 같습니다.
   >
   >* 트리에서 랜딩 페이지를 마우스 오른쪽 단추로 클릭하고 를 선택합니다 **테스트 그룹으로 변환**
   >* 프로그램에서 다음을 선택하여 랜딩 페이지 테스트 그룹 만들기 **AB 테스트 그룹** 에서 **새 로컬 자산** 메뉴


1. 이름과 선택적 설명을 입력합니다. 테스트할 랜딩 페이지 를 선택하고 **만들기**.

   ![](assets/image2015-8-5-13-3a39-3a10.png)

   >[!NOTE]
   >
   >승인되지 않은 랜딩 페이지만 선택할 수 있습니다.

1. 각 랜딩 페이지를 마우스 오른쪽 단추로 클릭하고 를 선택합니다 **승인**.

   ![](assets/three-1.png)

1. 을(를) 클릭합니다. **테스트 그룹 옵션** 드롭다운 및 선택 **테스트 그룹 승인**.

   ![](assets/four-1.png)

   됐습니다. 이제 선택한 랜딩 페이지의 상태를 비교할 수 있습니다.

   ![](assets/five.png)

   >[!TIP]
   >
   >테스트 그룹을 삭제하려면 **그룹 작업 테스트** 을(를) 선택합니다. **테스트 그룹 삭제**.
