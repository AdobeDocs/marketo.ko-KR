# 이벤트 파트너 {#event-partners}

Marketo을 이벤트 파트너에 연결하여 온라인 이벤트를 자동화합니다. 연결할 때 다음을 수행할 수 있습니다.

* 웨비나 이벤트 관리
* Marketo 랜딩 페이지에서 캡처한 등록 정보를 구성된 이벤트 파트너에 푸시
* 출석 정보를 Marketo 이벤트로 자동 가져오기
* 프로그램의 구성원으로 사람들의 상태 캡처

온라인 Marketo 이벤트와 이벤트 파트너 통합에 대한 지침을 참조하십시오.
