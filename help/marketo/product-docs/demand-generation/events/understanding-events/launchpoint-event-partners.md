# 이벤트 파트너 {#event-partners}

Marketo을 이벤트 파트너에 연결하여 온라인 이벤트를 자동화합니다. 연결할 때 다음을 수행할 수 있습니다.

* 웨비나 이벤트 관리
* Marketo 랜딩 페이지에서 캡처한 푸시 등록 정보를 구성된 이벤트 파트너에게 보냅니다.
* Marketo 이벤트에 자동으로 참석 정보 가져오기
* 프로그램의 구성원으로 사람 상태 캡처

온라인 Marketo 이벤트와 이벤트 파트너를 통합하는 방법에 대한 지침을 참조하십시오.
