# LaunchPoint 이벤트 파트너 {#launchpoint-event-partners}

LaunchPoint 이벤트 파트너에 Marketing을 연결하여 온라인 이벤트를 자동화할 수 있습니다. 연결을 할 때 다음을 수행할 수 있습니다.

* 웨비나 이벤트 관리
* Marketing 랜딩 페이지에서 구성된 LaunchPoint 이벤트 파트너로 캡처한 푸시 등록 정보
* Marketing 이벤트에서 자동으로 참석 정보 가져오기
* 프로그램 멤버로 사용자의 상태 캡처

온라인 마케팅 이벤트를 LaunchPoint 이벤트 파트너와 통합하는 방법을 참조하십시오.
