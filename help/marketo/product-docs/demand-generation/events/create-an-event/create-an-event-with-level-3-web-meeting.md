# 수준 3 웹 모임으로 이벤트 만들기 {#create-an-event-with-level-web-meeting}

>[!PREREQUISITES]
>
>* [LaunchPoint Service로 수준 3 웹 모임 추가](/help/marketo/product-docs/administration/additional-integrations/add-level-3-web-meeting-as-a-launchpoint-service.md)
>* [새 이벤트 프로그램 만들기](/help/marketo/product-docs/demand-generation/events/understanding-events/create-a-new-event-program.md)
>* 적절한 설정 [흐름 작업](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/add-a-flow-step-to-a-smart-campaign.md)참여를 추적하려면

먼저 레벨 3에서 웨비나를 만듭니다. 도움이 필요한 경우 [레벨 3 리소스 라이브러리](https://www.level3.com/en/resource-library/). BrightTalk과 매우 비슷하다는 것을 알게 될 것입니다.  Marketo은 레벨 3 필드의 작은 하위 집합을 사용합니다.

* **이름** - 웹 캐스트의 이름입니다.
* **시작일** - 웹캐스트의 시작 날짜입니다.
* **종료일** - 웹캐스트의 종료 날짜입니다.
* **시간대** - 웹 캐스트에 설정된 표준 시간대입니다.
* **설명** - 웹캐스트 설명.

1. 새 이벤트를 선택합니다. 클릭 **이벤트 작업,** 그러면 **이벤트 설정.**

   ![](assets/image2016-3-24-15-3a40-3a39.png)

1. Event Partner 아래에서 **수준 3 웹 모임**.

   ![](assets/image2016-3-24-15-3a42-3a10.png)

1. 로그인 아래에서 레벨 3 로그인을 선택합니다.

   ![](assets/image2016-3-24-15-3a43-3a43.png)

1. 이벤트 아래에서 사용할 레벨 3 이벤트를 선택합니다.

   ![](assets/image2016-3-24-15-3a44-3a41.png)

1. 클릭 **저장**.

   ![](assets/image2016-3-24-15-3a45-3a31.png)

   최고입니다! 이벤트를 레벨 3에 연결했습니다!

## 일정 보기  {#viewing-the-schedule}

프로그램 일정 보기에서 이벤트의 달력 항목을 클릭합니다. 일정표는 화면 오른쪽에 있습니다!

![](assets/image2016-3-24-15-3a51-3a7.png)
