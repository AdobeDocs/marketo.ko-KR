# 수준 3 웹 모임을 사용하여 이벤트 만들기 {#create-an-event-with-level-web-meeting}

>[!PREREQUISITES]
>
>* [수준 3 웹 모임을 LaunchPoint 서비스로 추가](/help/marketo/product-docs/administration/additional-integrations/add-level-3-web-meeting-as-a-launchpoint-service.md)
>* [새 이벤트 프로그램 만들기](/help/marketo/product-docs/demand-generation/events/understanding-events/create-a-new-event-program.md)
>* 적절한 [흐름 작업](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/add-a-flow-step-to-a-smart-campaign.md)참여를 추적합니다


먼저 수준 3에서 웨비나를 만듭니다. 도움이 필요하면 다음을 확인하십시오 [레벨 3 리소스 라이브러리](https://www.level3.com/en/resource-library/). 여러분은 그것이 BrightTalk와 매우 유사하다는 것을 알게 될 것입니다.  Marketo에서는 수준 3 필드의 작은 하위 집합을 사용합니다.

* **이름** - 웹캐스트의 이름입니다.
* **시작 날짜** - 웹캐스트 시작 날짜입니다.
* **종료 날짜** - 웹캐스트 종료 날짜입니다.
* **시간대** - 웹캐스트에 대해 설정된 시간대입니다.
* **설명** - 웹캐스트 설명입니다.

1. 새 이벤트를 선택합니다. 클릭 **이벤트 작업,** 그런 다음 **이벤트 설정.**

   ![](assets/image2016-3-24-15-3a40-3a39.png)

1. 이벤트 파트너에서 을 선택합니다. **레벨 3 웹 모임**.

   ![](assets/image2016-3-24-15-3a42-3a10.png)

1. 로그인 아래에서 수준 3 로그인을 선택합니다.

   ![](assets/image2016-3-24-15-3a43-3a43.png)

1. 이벤트 아래에서 사용할 레벨 3 이벤트를 선택합니다.

   ![](assets/image2016-3-24-15-3a44-3a41.png)

1. 클릭 **저장**.

   ![](assets/image2016-3-24-15-3a45-3a31.png)

   가장 훌륭합니다! 이벤트를 수준 3에 연결했습니다!

## 일정 보기  {#viewing-the-schedule}

프로그램 예약 보기에서 이벤트의 달력 항목을 클릭합니다. 화면 오른쪽에는 일정이 보입니다!

![](assets/image2016-3-24-15-3a51-3a7.png)
