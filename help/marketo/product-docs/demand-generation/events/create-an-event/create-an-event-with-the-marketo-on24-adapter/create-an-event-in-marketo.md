# Marketing에서 이벤트 만들기 {#create-an-event-in-marketo}

마케팅 이벤트는 프로그램을 통해 사람들의 진행 상황을 추적합니다. ON24 어댑터를 사용하여 등록 정보를 푸시하고 참석 정보를 가져옵니다. 이 이벤트는 진행 중인 사람들의 상태를 캡처합니다.

데이터 교환 방법은 다음과 같습니다.

![](assets/image2015-12-16-13-33-56.png)

마케팅 이벤트를 만들 때 채널 유형으로 **웨비나를** 선택합니다. 관리에서 이 채널을 편집하고 새 채널을 만들 수 있습니다. 새 채널을 만드는 경우 통합이 작동하려면 **웨비나가** 있는 이벤트 유형이어야 합니다. 자세한 [내용은 프로그램 채널](../../../../../product-docs/administration/tags/create-a-program-channel.md) 만들기를 참조하십시오.

다음 단계는 이벤트 설정을 [구성하고 웨비나와 Marketing to를 동기화하는 것입니다](https://docs.marketo.com/x/IRCa).
