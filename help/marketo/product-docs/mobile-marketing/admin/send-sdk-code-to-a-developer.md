# 개발자 {#send-sdk-code-to-a-developer}에게 SDK 코드 보내기

인앱 메시지 또는 푸시 알림을 만들려면 먼저 개발자가 Android 및 iOS 플랫폼용 모바일 앱 SDK를 설정하고 초기화하도록 해야 합니다.

* [Android용 지침](http://developers.marketo.com/documentation/mobile/installation-instructions-on-android/)
* [iOS용 지침](http://developers.marketo.com/documentation/mobile/installation-instructions-on-ios/)

## 개발자 {#send-sdk-code-to-a-developer-1}에게 SDK 코드 보내기

관리자가 일부 SDK 코드를 개발자에게 보내야 하는 경우가 있습니다.

![](assets/image2016-3-9-16-3a24-3a14.png)

방법 살펴보기

1. 관리자를 클릭합니다.

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. 모바일 앱을 선택합니다.

   ![](assets/image2015-4-22-16-3a14-3a29.png)

1. 원하는 모바일 앱을 선택합니다.

   ![](assets/image2015-4-22-16-3a33-3a19.png)

1. **모바일 앱 작업**&#x200B;을 클릭하고 **개발자에게 보내기**&#x200B;를 선택합니다.

   ![](assets/image2015-4-22-17-3a13-3a30.png)

1. 이메일 주소를 입력하고 **SEND**&#x200B;를 클릭합니다.

   ![](assets/image2015-4-22-18-3a51-3a54.png)

   이제 SDK 코드가 개발자에게 전송됩니다.
