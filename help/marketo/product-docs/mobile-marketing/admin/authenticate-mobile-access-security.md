# 모바일 액세스 보안 인증 {#authenticate-mobile-access-security}

모바일 사용자에 대한 보안을 개선하기 위해 Marketo에서는 두 개의 새 ID를 제공합니다. 액세스 키 및 액세스 비밀입니다. 이러한 기능은 앱에 로그인하는 사용자가 실제로 표시된 사용자인지 확인하는 데 도움이 됩니다.

ID를 사용하려면 코드를 획득하고 IT 부서가 보안 일치를 설정할 수 있도록 해당 코드를 제공해야 합니다.

>[!PREREQUISITES]
>
>이 기능을 사용하려면 지원 센터에 문의하여 &quot;Vespa 보안 모드&quot;를 요청하십시오.

1. Marketo에서 **관리** 을(를) 클릭합니다. **모바일 앱 및 장치**.

   ![](assets/image2015-12-1-14-3a36-3a30.png)

1. 보안 액세스를 활성화할 모바일 앱을 선택합니다.

   ![](assets/image2015-12-2-10-3a18-3a6.png)

1. 아래로 스크롤하여 액세스 보안 섹션에서 **편집**.

   ![](assets/image2015-12-1-14-3a41-3a37.png)

1. 을(를) 확인합니다. **보안 사용** 확인란을 선택합니다. 클릭 **저장**.

   ![](assets/image2015-12-1-14-3a54-3a0.png)

1. 이제 액세스 보안 영역에 두 개의 새 코드가 표시됩니다. IT 부서에 보내 주십시오.

   ![](assets/image2015-12-1-14-3a57-3a34.png)
