# 리드 가져오기 {#import-leads}

필드 매핑 지원을 통해 리드 목록을 Marketo Engage 데이터베이스로 가져오고 중복 제거합니다.

리드 목록을 가져오시겠습니까? 사람? csv 형식으로 일부 데이터 표준화 규칙을 적용하고 Marketo 데이터베이스에 저장합니다.

이 에이전트를 클릭하면 대화형 AI로 이동합니다. 왼쪽 창에는 에이전트가 실행할 데이터 표준화 기능에 대한 지침, 응답 및 옵션을 게시할 수 있는 창을 제공하는 AI 도우미가 있습니다.

가져오기 프로세스를 시작하려면 첨부 파일 아이콘을 클릭하여 파일을 업로드합니다.

가져오기 목록을 입력하고 보내기 를 클릭합니다

적용할 비즈니스 규칙을 입력하고 [보내기]를 클릭합니다. 입력하면 중앙 콘솔의 창에서 적용된 규칙의 결과를 제공합니다.

원하는 경우 다른 비즈니스 규칙을 입력하고 보내기를 누릅니다.

데이터베이스에 매핑된 목록 필드 목록을 보려면 매핑 탭을 클릭합니다.

데이터베이스에 목록을 저장할 준비가 되면 Marketo으로 가져오기 를 클릭합니다.

목록을 저장할 폴더를 선택하고 리드 목록의 이름을 입력한 다음 동의 상자를 선택한 다음 &quot;Marketo으로 승인 및 가져오기&quot; 단추를 클릭합니다.

목록이 저장되면 처리된 레코드, 실패한 레코드 또는 경고가 있는 레코드의 요약이 확인에 제공됩니다.
