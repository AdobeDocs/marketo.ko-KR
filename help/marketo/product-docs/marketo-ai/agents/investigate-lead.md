# 리드 조사 {#investigate-lead}

누군가 MQL을 사용하지 않았거나, 프로그램에 적격이거나, 라이프사이클을 진행하지 않은 이유를 알아보십시오.

조사 시나리오의 전체 목록을 보려면 오른쪽에 있는 아래쪽 화살표를 클릭합니다.

스크린샷

| **Title** | 설명 |
|----------|------------|
| **MQL 조사** | 특정 잠재 고객이 자격을 얻지 못하거나, 참여를 중단하거나, 중단된 이유에 대한 세부 정보를 제공합니다. |
| **프로그램 자격** | 잠재 고객이 육성 프로그램에 적합하지 않은 이유에 대한 세부 정보를 제공합니다. |
| **주기 진행률** | Lead 가 예상보다 오래 스테이지에 머무르는 이유를 탐색하는 데 사용됩니다. |
| **참여 분석** | 잠재 고객이 이메일 열기 또는 클릭을 중단한 경우 사용됩니다. |
| **캠페인 여정** | 잠재 고객의 전체 경로 및 여정 터치포인트를 보는 데 사용됩니다. |
| **데이터 품질** | 잘못된 데이터가 잠재 고객에 문제를 일으킬 가능성이 있을 때 사용됩니다. |
| **판매 전달** | Sales에서 Lead 를 거부하고 후속 작업이 없는 경우 사용됩니다. |
| **속성** | 영업 기회 생성을 유도한 잠재 고객에 영향을 미친 영향을 탐색하는 데 사용됩니다. |
| **점수 감소** | 잠재 고객 스코어가 예기치 않게 떨어진 이유를 조사하려면 |

>[!TIP]
>
>9개의 시나리오 목록을 모두 빠르게 보려면 맨 위의 프롬프트 창에 `/lead investigation`을(를) 입력하십시오.
