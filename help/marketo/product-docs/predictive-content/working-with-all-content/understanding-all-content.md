# 모든 컨텐츠 이해 {#understanding-all-content}

모든 컨텐츠 페이지에는 수동으로 검색되거나 추가된 모든 컨텐츠가 표시됩니다.

![](assets/image2017-10-3-9-3a4-3a56.png)

페이지 필드에는 다음이 포함됩니다.

* **제목**: 컨텐츠의 이름
* **카테고리**: 사용자가 생성했으며 웹 또는 이메일에 대한 예측 결과를 그룹화하는 데 사용됩니다
* **보기 횟수**: 검색된 웹 컨텐츠(모든 소스 포함)에 대한 총 클릭 수
* **전환**: 동일한 방문에서 컨텐츠 조각을 보고 양식을 완료한 방문자의 수입니다
* **전환율**: 직접 전환으로 계산된 비율을 클릭 수로 나눈 값입니다
* **예측 승인**: 컨텐츠가 승인되면 아이콘이 표시됩니다

새 콘텐츠를 추가하고 추적하거나, 콘텐츠를 제거하거나, 콘텐츠 일부를 승인할 수 있습니다. 컨텐츠를 승인한 후 예측 컨텐츠 페이지에서 편집할 수 있습니다.

## 카테고리별 컨텐츠 필터링  {#filter-content-by-categories}

하나 이상의 카테고리를 선택하여 콘텐츠를 필터링합니다. 그런 다음 표준 또는 사용자 지정 설정을 사용하여 컨텐츠 분석에 대한 날짜 또는 날짜 범위를 설정합니다.

1. 설정 **모든 컨텐츠** 페이지에서 필터 아이콘을 클릭하고 **카테고리**, 컨텐츠 카테고리를 한 개 이상 선택합니다.

   ![](assets/image2017-10-3-9-3a5-3a52.png)

1. 이제 해당 카테고리의 콘텐츠만 표시됩니다.

   ![](assets/image2017-10-3-9-3a6-3a23.png)

## 날짜 또는 날짜 범위에 대한 Analytics 표시 {#display-analytics-for-a-date-or-date-range}

1. 특정 날짜 또는 날짜 범위에 대한 데이터를 표시하려면 달력 아이콘을 클릭한 다음 왼쪽 패널에서 를 선택합니다. 사용자 지정 날짜 범위의 경우 달력(표시된 대로)에서 날짜 범위의 첫 번째 및 마지막 날을 클릭합니다. 클릭 **적용**.

   ![](assets/all-content-calendar-filter-hands.png)

1. 이제 선택한 날짜 범위의 콘텐츠에 대해 analytics가 표시됩니다.
