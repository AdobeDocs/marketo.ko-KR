# 이메일 HTML 다운로드 {#download-an-emails-html}

백업 및 다른 목적으로 Marketo에서 이메일의 HTML 컨텐츠를 다운로드할 수 있습니다.

1. 이메일을 찾아 선택합니다. 에서 **이메일 작업** 드롭다운 클릭 **다운로드 HTML**.

   ![](assets/one-4.png)

그렇게 쉬워요!
