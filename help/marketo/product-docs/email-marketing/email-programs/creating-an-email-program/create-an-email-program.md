# 이메일 프로그램 만들기 {#create-an-email-program}

이메일 프로그램을 사용하여 사용자 그룹에 이메일을 빠르고 쉽게 보낼 수 있습니다.

1. 이동 **마케팅 활동**.

   ![](assets/one.png)

1. 에서 프로그램을 만들 폴더를 선택하고 **새로 만들기** 드롭다운을 선택하고 을(를) 선택합니다. **새 프로그램**.

   ![](assets/two.png)

1. 이름을 입력하고, **이메일** 를 프로그램 유형으로 선택하고 **만들기**.

   ![](assets/three.png)

   >[!NOTE]
   >
   >선택 시 **이메일** 프로그램 유형으로 채널은 자동으로 **이메일 보내기**. 원한다면 변경할 수 있습니다.

   ![](assets/four.png)

좋아! 이제 프로그램이 트리에 있고 사용할 준비가 되었습니다. 다음 단계는 대상자를 정의하는 것입니다. 아래의 Marketo 관련 문서를 참조하십시오.
