# &quot;제목&quot; A/B 테스트 사용 {#use-subject-line-a-b-testing}

A/B에서 이메일을 쉽게 테스트할 수 있습니다. 가장 일반적인 테스트 중 하나는 **제목 라인** 테스트입니다.

>[!NOTE]
>
>**사전 요구 사항**
>
>* [A/B 테스트 추가](add-an-a-b-test.md)

>



1. 이메일 타일에서 이메일이 선택된 상태로 A/B 테스트 추가를 클릭합니다.

![](assets/image2014-9-12-15-3a6-3a2.png)

1. 테스트 편집기 창이 열립니다. 하나 이상의 새 제목 라인을 입력합니다.

   >[!NOTE]
   >
   >선택*** A*****선택한 이메일에 포함된 정보가 미리 채워집니다.

   ![](assets/image2014-9-12-15-3a9-3a14.png)

   >[!TIP]
   >
   >더하기** **기호****를 클릭하여 제목 줄을 더 추가할 수 있습니다.

1. 슬라이더를 사용하여 A/B 테스트를 받을 대상자의 백분율을 선택하고 **다음을 클릭합니다**.

   ![](assets/image2014-9-12-15-3a10-3a4.png)

   >[!CAUTION]
   >
   >**샘플 크기를 100%로 설정하지 않는 것이 좋습니다**. 정적 목록을 사용하는 경우 샘플 크기를 100%로 설정하면 모든 사용자에게 이메일이 전송되며 우승자는 아무도 전송하지 않습니다. 스마트 목록을 사용하는 경우 샘플 크기를 100%로 설정하면 해당 시간에 모든 사용자에게 이메일을 보냅니다. *그리고 이메일 프로그램이 나중에 다시 실행될 때 스마트 목록에 자격이 되는 모든 새로운 사람도 현재 대상에 포함되기 때문에 이메일을 받게 됩니다.

   >[!NOTE]
   >
   >다양한 제목 변경 작업은 선택된 테스트 샘플 크기의 일부까지 가져옵니다.

   좋아, 거의 다 왔어 이제 A/B 테스트 [승자 기준을 정의해야 합니다](define-the-a-b-test-winner-criteria.md).
