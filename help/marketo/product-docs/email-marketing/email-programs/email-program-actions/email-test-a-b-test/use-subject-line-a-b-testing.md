# &quot;제목란&quot; A/B 테스트 사용 {#use-subject-line-a-b-testing}

A/B가 이메일을 쉽게 테스트할 수 있습니다. 가장 일반적인 테스트 중 하나는 **제목 줄** 테스트.

>[!PREREQUISITES]
>
>[A/B 테스트 추가](/help/marketo/product-docs/email-marketing/email-programs/email-program-actions/email-test-a-b-test/add-an-a-b-test.md)

1. 이메일 타일에서 이메일을 선택한 채 A/B 테스트 추가를 클릭합니다.

![](assets/image2014-9-12-15-3a6-3a2.png)

1. 테스트 편집기 창이 열립니다. 새 제목 줄을 하나 이상 입력합니다.

   >[!NOTE]
   >
   >선택 **A** 은(는) 선택한 이메일에 포함된 정보로 미리 채웁니다.

   ![](assets/image2014-9-12-15-3a9-3a14.png)

   >[!TIP]
   >
   >을(를) 클릭합니다. **+** 제목 줄을 더 추가하려면

1. 슬라이더를 사용하여 A/B 테스트를 받을 대상자의 비율을 선택하고 을 클릭합니다 **다음**.

   ![](assets/image2014-9-12-15-3a10-3a4.png)

   >[!CAUTION]
   >
   >**샘플 크기를 100%로 설정하지 않는 것이 좋습니다**. 정적 목록을 사용하는 경우 샘플 크기를 100%로 설정하면 대상이 있는 모든 사람에게 이메일이 전송되고 승자는 아무도 표시되지 않습니다. 스마트 목록을 사용하는 경우 샘플 크기를 100%로 설정하면 해당 이메일이 대상자의 모든 사람에게 전송됩니다 _해당 시간_. 그리고 이메일 프로그램이 나중에 다시 실행되면 스마트 목록에 대한 자격이 있는 새로운 사람도 이제 대상에 포함되므로 이메일을 받게 됩니다.

   >[!NOTE]
   >
   >다양한 제목 변형은 선택한 테스트 샘플 크기 부분을 사용합니다.

   거의 다 왔어 이제 우리는 [A/B 테스트 승자 기준 정의](/help/marketo/product-docs/email-marketing/email-programs/email-program-actions/email-test-a-b-test/define-the-a-b-test-winner-criteria.md).
