# &quot;보낸 사람 주소&quot; A/B 테스트 사용 {#use-from-address-a-b-testing}

A/B가 이메일을 쉽게 테스트할 수 있습니다. 한 가지 흥미로운 테스트는 **보낸 사람 주소** 테스트. 설정 방법은 다음과 같습니다.

>[!PREREQUISITES]
>
>[A/B 테스트 추가](/help/marketo/product-docs/email-marketing/email-programs/email-program-actions/email-test-a-b-test/add-an-a-b-test.md)

1. 아래에 **이메일** 타일, 이메일을 선택한 상태로 **A/B 테스트 추가**.

   ![](assets/image2014-9-12-15-3a32-3a8.png)

1. 새 창이 열리면 다음을 선택합니다 **보낸 사람 주소** 대상 **테스트 유형**.

   ![](assets/image2014-9-12-15-3a32-3a22.png)

1. 이전 테스트 정보(예: 제목 테스트)가 있는 경우 안전하게 **테스트 재설정**.

   ![](assets/image2014-9-12-15-3a32-3a28.png)

1. 두 번째 **보낸 사람 주소** 테스트할 정보입니다.

   >[!NOTE]
   >
   >선택 A는 선택한 이메일에 포함된 정보로 미리 채워집니다.

   ![](assets/image2014-9-12-15-3a32-3a34.png)

   >[!TIP]
   >
   >을(를) 클릭합니다. **+** 보낸 사람 주소를 원하는 만큼 추가하려면

1. 슬라이더를 사용하여 A/B 테스트에서 원하는 대상 비율을 선택하고 을 클릭합니다 **다음**.

   ![](assets/image2014-9-12-15-3a33-3a41.png)

   >[!NOTE]
   >
   >다른 변형은 선택한 테스트 샘플 크기의 동일한 부분으로 전송됩니다.

   >[!CAUTION]
   >
   >**샘플 크기를 100%로 설정하지 않는 것이 좋습니다**. 정적 목록을 사용하는 경우 샘플 크기를 100%로 설정하면 대상자가 모든 사람에게 이메일이 전송되고 승자는 아무도 표시되지 않습니다. 사용 중인 **smart** 목록, 샘플 크기를 100%로 설정하면 해당 이메일이 대상자의 모든 사람에게 전송됩니다 _해당 시간_. 이메일 프로그램이 나중에 다시 실행되면 스마트 목록에 대한 자격이 있는 새로운 사람도 이제 대상에 포함되므로 이메일을 받게 됩니다.

   거의 다 왔어요 이제 우리는 [A/B 테스트 승자 기준 정의](/help/marketo/product-docs/email-marketing/email-programs/email-program-actions/email-test-a-b-test/define-the-a-b-test-winner-criteria.md).
