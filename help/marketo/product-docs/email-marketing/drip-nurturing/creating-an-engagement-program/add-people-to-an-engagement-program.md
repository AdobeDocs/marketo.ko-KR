# 참여 프로그램에 사용자 추가 {#add-people-to-an-engagement-program}

사람들이 참여 프로그램의 구성원이 될 수 있는 방법에는 여러 가지가 있습니다.

* [[!UICONTROL Add to Engagement Program]](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/add-to-engagement-program.md)
* [[!UICONTROL Change Engagement Program Cadence]](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/change-engagement-program-cadence.md) 흐름 단계
* [[!UICONTROL Change Engagement Program Stream]](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/change-engagement-program-stream.md) 흐름 단계
* [[!UICONTROL Change Program Status]](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/change-program-status.md) 흐름 단계
* 사용자가 참여 프로그램의 로컬 양식을 작성하는 경우
* 참여 프로그램으로 구성원 가져오기(프로그램 내의 구성원 탭을 통해)
* [!DNL Salesforce]을(를) CRM으로 사용하고 있으며 참여 프로그램에 [!DNL Salesforce] 캠페인이 연결되어 있는 경우 CRM측의 [!DNL Salesforce] 캠페인에 사용자를 추가하면 해당 사용자도 참여 프로그램의 구성원이 됩니다.
