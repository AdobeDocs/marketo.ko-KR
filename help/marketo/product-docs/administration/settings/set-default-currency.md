# 기본 통화 설정 {#set-default-currency}

Marketo Engage 구독에 대한 기본 통화를 보고 편집하는 방법을 알아봅니다.

>[!NOTE]
>
>**관리자 권한이 필요합니다.**

1. **[!UICONTROL Admin]** 영역으로 이동합니다.

   ![](assets/set-default-currency-1.png)

1. **[!UICONTROL Location]**&#x200B;를 클릭합니다.

   ![](assets/set-default-currency-2.png)

1. **[!UICONTROL Edit]**&#x200B;에서 [!UICONTROL Subscription Currency Settings]을(를) 클릭합니다.

   ![](assets/set-default-currency-3.png)

1. 선택한 통화 형식을 선택하고 **[!UICONTROL Save]**&#x200B;을(를) 클릭합니다.

   ![](assets/set-default-currency-4.png)
