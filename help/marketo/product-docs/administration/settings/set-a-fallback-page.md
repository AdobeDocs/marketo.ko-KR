# 대체 페이지 설정 {#set-a-fallback-page}

대체 페이지는 랜딩 페이지가 오프라인이거나 찾을 수 없는 경우 마지막 방어선입니다. 꼭 하나 갖고 있는지 확인하세요.

>[!NOTE]
>
>**관리 권한 필요**

1. 로 이동합니다. **관리** 영역.

   ![](assets/set-a-fallback-page-1.png)

1. 클릭 **랜딩 페이지**.

   ![](assets/set-a-fallback-page-2.png)

1. 아래에 **랜딩 페이지** 탭, **편집**.

   ![](assets/set-a-fallback-page-3.png)

1. 을(를) 입력합니다. **대체 페이지** 대화 상자에서 를 클릭하고 **저장**.

   ![](assets/set-a-fallback-page-4.png)

불라! 방금 대체 페이지를 추가했습니다. 404페이지 더 이상 없음.
