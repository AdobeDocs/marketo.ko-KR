# 최소 시스템 요구 사항 {#minimum-system-requirements}

로그인하고 Marketo을 사용해야 하는 것이 무엇인지 궁금하다면 찾고 있는 정보는 다음과 같습니다.

>[!NOTE]
>
>iOS 및 Android는 아직 지원되지 않습니다.

* Windows, OS X 또는 Linux를 실행하는 컴퓨터 또는 노트북
* 키보드 및 마우스
* 최소 해상도가 1280 x 768인 모니터
* 최신 브라우저인 경우 [지원되는 브라우저](/help/marketo/product-docs/administration/setup-administration/supported-browsers.md)자세한 내용은 target=&quot;_blank&quot;}
* 인터넷 연결

>[!NOTE]
>
>IT 팀이를 사용하여 웹 액세스를 제한하는 허용 목록에 추가하다 경우 추가하도록 요청하십시오 `*.marketo.com` 모든 Marketo 리소스 및 웹 소켓을 허용합니다.
