# API 전용 사용자 역할 만들기 {#create-an-api-only-user-role}

REST API를 통해 Marketing [](http://developers.marketo.com/documentation/rest/)와 통합하려는 경우 사용자를 API 전용 역할 [으로](create-an-api-only-user.md) 지정한 다음 ReST API와 함께 사용할 사용자 지정 서비스를 [만드는 것이 좋습니다](../../../product-docs/administration/additional-integrations/create-a-custom-service-for-use-with-rest-api.md). 다음은 API 전용 사용자 역할을 만드는 방법입니다.

>[!NOTE]
>
>**관리자 권한 필요**

1. 내 마켓에서 관리를 **클릭합니다**.

   ![](assets/adminhand-1.png)

1. 보안 아래에서 **사용자 및 역할을 클릭합니다**.

   ![](assets/two.png)

1. **Roles **탭을 클릭한 다음 **새 역할을 클릭합니다**.

   ![](assets/image2014-9-16-13-3a47-3a12.png)

1. 역할 이름을 입력하고 부여할 API 액세스 권한을 선택한 다음 **만들기를 클릭합니다.**

   ![](assets/image2014-9-16-13-3a47-3a36.png)

>[!TIP]
>
>필요한 최소 권한만 부여하는 것이 좋습니다.

>[!MORELIKETHIS]
>
>좋아, 잘했다! 이제 API [전용 사용자를 만듭니다](create-an-api-only-user.md).
