# 프로그램 채널에서 프로그램 상태 삭제 {#delete-a-program-status-from-a-program-channel}

프로그램 상태는 프로그램 경로(채널)를 통한 체크포인트입니다. 실수로 또는 더 이상 필요하지 않은 상태를 만든 경우 자유롭게 삭제하십시오.

1. 로 이동합니다. **관리** 영역.

   ![](assets/delete-a-program-status-from-a-program-channel-1.png)

1. 클릭 **태그**.

   ![](assets/delete-a-program-status-from-a-program-channel-2.png)

1. 상태를 제거할 채널을 선택하고 **태그 작업**&#x200B;를 클릭합니다. **편집**.

   ![](assets/delete-a-program-status-from-a-program-channel-3.png)

1. 빨간색 클릭 **X** 상태를 제거하려면 **저장**.

   ![](assets/delete-a-program-status-from-a-program-channel-4.png)

   >[!TIP]
   >
   >현재 해당 상태에 지정된 사람이 있으면 삭제할 수 없지만 숨길 수 있습니다.
