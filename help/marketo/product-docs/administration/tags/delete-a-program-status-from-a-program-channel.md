# 프로그램 채널에서 프로그램 상태 삭제 {#delete-a-program-status-from-a-program-channel}

프로그램 상태는 프로그램 경로(채널)를 통한 체크포인트입니다. 실수로 또는 더 이상 필요하지 않은 상태를 만든 경우 자유롭게 삭제하십시오.

1. 로 이동합니다. **관리** 섹션을 참조하십시오.

   ![](assets/admin.png)

1. 클릭 **태그**.

   ![](assets/image2014-9-24-15-3a51-3a24.png)

1. 상태를 제거할 채널을 선택하고 **태그 작업**&#x200B;를 클릭합니다. **편집**.

   ![](assets/image2014-9-24-15-3a51-3a45.png)

1. 클릭 ![X 아이콘](assets/image2014-9-24-15-3a52-3a39.png) 상태를 제거하려면 **저장**.

   >[!TIP]
   >
   >현재 해당 상태에 지정된 사람이 있으면 삭제할 수 없지만 숨길 수 있습니다.

   ![](assets/image2014-9-24-15-3a57-3a53.png)
