# 보고서 이메일의 URL 만료 시간 변경 {#change-the-expiration-time-for-urls-in-report-emails}

>[!NOTE]
>
>**관리 권한 필요**

보고서 구독 이메일의 링크는 3일 후에 만료됩니다. 이러한 링크의 만료 시간을 변경하려면 다음 단계를 따르십시오.

1. 아래 **관리**&#x200B;를 클릭하고 **로그인 설정**.

   ![](assets/image2014-9-16-14-3a44-3a57.png)

1. 을(를) 클릭합니다. **URL 만료 편집** 버튼을 클릭합니다.

   ![](assets/image2014-9-16-14-3a45-3a1.png)

1. 드롭다운 메뉴에서 링크가 만료될 때까지 몇 일을 선택합니다. 클릭 **저장**.

   ![](assets/image2014-9-16-14-3a45-3a5.png)

   좋아요. 이메일 링크 만료 설정을 편집했습니다.

   >[!NOTE]
   >
   >이 기능은 마케팅 이메일이 아닌 보고서 및 경고의 링크에만 적용됩니다.
