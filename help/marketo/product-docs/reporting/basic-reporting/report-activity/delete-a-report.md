# 보고서 삭제 {#delete-a-report}

시작하면 [보고서 만들기](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md)이렇게 하면 많은 것을 얻을 수 있습니다. 더 이상 필요하지 않은 보고서를 삭제해야 합니다.

1. 트리에서 불필요한 보고서를 마우스 오른쪽 단추로 클릭하고 를 선택합니다 **보고서 삭제**.

   ![](assets/image2014-9-16-14-3a26-3a48.png)

1. 보고서를 삭제할지 확인합니다.

   ![](assets/image2014-9-16-14-3a26-3a53.png)

   보고서가 트리에서 사라집니다. 지금 몇 가지 더 오래된 보고서를 제거하십시오!
