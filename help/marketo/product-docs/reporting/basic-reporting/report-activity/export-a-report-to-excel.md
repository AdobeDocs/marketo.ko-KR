# 보고서를 Excel로 내보내기 {#export-a-report-to-excel}

보고서를 Excel 파일로 내보내 다른 소프트웨어에서 데이터를 사용할 수 있습니다.

1. 마케팅 활동 **영역으로** 이동합니다.

   ![](assets/image2014-9-16-13-3a11-3a14.png)

1. 탐색 트리에서 보고서를 선택하고 **보고서** 탭을 클릭합니다.

   ![](assets/image2014-9-16-13-3a11-3a18.png)

1. 내보내기 **단추를** 클릭합니다.

   ![](assets/image2014-9-16-13-3a11-3a21.png)

   그게 다예요 브라우저에 Excel 파일을 시스템에 저장하라는 메시지가 표시됩니다.

   >[!NOTE]
   >
   >**관련 문서**
   >
   >
   >다운로드한 파일이 너무 크면 보고서 크기를 [변경할 수 있습니다](../../../../product-docs/reporting/basic-reporting/editing-reports/configure-report-size.md) .

   >[!NOTE]
   >
   >**딥 다이브**
   >
   >
   >기본 보고 [심층 분석에서 보고서를 사용하여 수행할 수 있는 다른 유용한](http://docs.marketo.com/display/docs/basic+reporting) 모든 기능을 살펴볼 수 있습니다.
