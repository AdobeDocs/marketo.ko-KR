# Excel로 보고서 내보내기 {#export-a-report-to-excel}

보고서를 Excel 파일로 내보내 다른 소프트웨어의 데이터로 작업할 수 있습니다.

1. 로 이동 **마케팅 활동** 영역입니다.

   ![](assets/image2014-9-16-13-3a11-3a14.png)

1. 탐색 트리에서 보고서를 선택하고 **보고서** 탭.

   ![](assets/image2014-9-16-13-3a11-3a18.png)

1. 다음을 클릭합니다. **내보내기** 단추를 클릭합니다.

   ![](assets/image2014-9-16-13-3a11-3a21.png)

   그게 다야! Excel 파일을 시스템에 저장하라는 메시지가 브라우저에 표시됩니다.

   >[!MORELIKETHIS]
   >
   >다운로드한 파일이 너무 크면 다음을 수행할 수 있습니다. [보고서 크기 변경](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/configure-report-size.md).
