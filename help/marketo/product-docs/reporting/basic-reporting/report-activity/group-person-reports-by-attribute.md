# 속성별 개인 보고서 그룹화 {#group-person-reports-by-attribute}

개인 보고서를 개인 또는 회사 속성별로 그룹화할 수 있습니다.

1. 로 이동합니다. **마케팅 활동** 또는 **Analytics**) 영역 내에 있어야 합니다.

   ![](assets/image2017-3-28-10-3a22-3a53.png)

1. 탐색 트리에서 개인 보고서를 선택하고 **설정** 탭.

   ![](assets/image2017-3-28-11-3a33-3a48.png)

1. 두 번 클릭 **그룹화 기준**.

   ![](assets/image2017-3-28-11-3a34-3a5.png)

   >[!NOTE]
   >
   >다음을 수행할 수도 있습니다 [세그먼트별로 개인 보고서 그룹화](/help/marketo/product-docs/personalization/segmentation-and-snippets/segmentation/group-person-reports-by-segment.md).

   사람 그룹 대화 상자에서 그룹화에 사용할 개인 또는 회사 속성을 선택합니다.

   ![](assets/image2017-3-28-11-3a34-3a42.png)

   >[!TIP]
   >
   >다음과 같이 숫자 값이 있는 속성을 선택하는 경우 _작성 위치_ 또는 _연간 매출_&#x200B;에서 지표를 선택합니다. **판매량** 오른쪽에 드롭다운.

   됐습니다. 보고서 탭을 클릭하여 적절하게 그룹화된 보고서를 확인합니다.

   ![](assets/image2017-3-28-11-3a35-3a0.png)
