# 작업 영역 간에 모델 공유 {#share-a-model-across-workspaces}

Marketing에서는 작업 영역 간에 모델을 공유할 수 있습니다. 방법

1. 분석 섹션으로 **이동합니다** .

   ![](assets/analytics.png)

1. My Models **폴더를** 마우스 오른쪽 단추로 클릭하고 **새 폴더를 클릭합니다**.

   ![](assets/image2014-10-3-14-3a5-3a23.png)

1. 폴더 이름을 지정합니다.

   ![](assets/image2014-10-3-14-3a5-3a38.png)

1. 공유할 모델을 **공유 폴더로 드래그합니다**.

   ![](assets/image2014-10-3-14-3a5-3a52.png)

1. 폴더를 마우스 오른쪽 단추로 클릭하고 폴더 **공유를 클릭합니다**.

   ![](assets/image2014-10-3-14-3a6-3a9.png)

   >[!NOTE]
   >
   >
   >다른 작업 공간과 모델을 공유하면 해당 사용자가 모델을 기반으로 보고서를 실행할 수 있습니다.

1. 폴더를 공유할 작업 영역을 선택하고 저장을 **클릭합니다**.

   ![](assets/image2014-10-3-14-3a6-3a22.png)

정말 쉬워! 이제 다른 작업 영역의 사용자가 공유 모델을 통해 이동할 수 있습니다. 해당 작업 공간에서 개인 지정을 실행할 수 있습니다. 자세한 내용은 [매출 주기 모델](https://community.marketo.com/MarketoArticle?id=kA050000000KyvQCAS) 시작을 참조하십시오.
