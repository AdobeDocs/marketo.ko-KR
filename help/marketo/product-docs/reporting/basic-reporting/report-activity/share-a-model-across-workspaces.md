# 작업 공간에서 모델 공유 {#share-a-model-across-workspaces}

Marketo은 작업 공간에서 모델을 공유할 수 있는 기능을 제공합니다. 방법은 다음과 같습니다.

1. 로 이동합니다. **Analytics** 섹션을 참조하십시오.

   ![](assets/analytics.png)

1. 마우스 오른쪽 단추를 클릭합니다. **내 모델** 폴더를 클릭하고 **새 폴더**.

   ![](assets/image2014-10-3-14-3a5-3a23.png)

1. 폴더 이름을 지정합니다.

   ![](assets/image2014-10-3-14-3a5-3a38.png)

1. 공유할 모델을 **폴더 공유**.

   ![](assets/image2014-10-3-14-3a5-3a52.png)

1. 폴더를 마우스 오른쪽 단추로 클릭하고 **폴더 공유**.

   ![](assets/image2014-10-3-14-3a6-3a9.png)

   >[!NOTE]
   >
   >모델을 다른 작업 공간과 공유하면 해당 사용자가 모델을 기반으로 보고서를 실행할 수 있습니다.

1. 폴더를 공유할 작업 영역을 선택하고 을 클릭합니다 **저장**.

   ![](assets/image2014-10-3-14-3a6-3a22.png)

그렇게 쉬워요! 이제 다른 작업 공간의 사람들이 공유 모델을 통해 이동할 수 있습니다.
