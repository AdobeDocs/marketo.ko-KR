# 캠페인 활동 보고서 {#campaign-activity-report}

사용 방법 보기 [스마트 캠페인](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/understanding-batch-and-trigger-smart-campaigns.md) 수행 중입니다.

1. [보고서 만들기](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md) 을(를) 선택하고 을(를) 선택합니다. **캠페인 활동** [보고서 유형](/help/marketo/product-docs/reporting/basic-reporting/report-types/report-type-overview.md).

1. [보고서 기간 설정](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/change-a-report-time-frame.md) 을 클릭하고 **보고서** 탭.

1. 좋아요! 보고서를 탐색하여 캠페인 유형별로 그룹화된 가장 효과적인 캠페인을 식별합니다.

   ![](assets/image2014-9-16-16-3a8-3a45.png)

   >[!TIP]
   >
   >예를 들어 가장 많은 사람을 처리한 캠페인을 찾으려면 [보고서 정렬](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/sort-report-on-columns.md) on _처리된 총 사용자 수_ 및 **내림차순 정렬**.

   스마트 캠페인의 이름을 클릭하여 엽니다.  [선택할 수 있는 열](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/select-report-columns.md) 캠페인 활동 보고서의 경우 다음을 포함합니다.

   | 열 | 설명 |
   |---|---|
   | 배치 실행 수 | 캠페인을 실행한 횟수입니다. |
   | 처리된 총 사용자 수 | 캠페인 흐름을 거친 사람 수입니다. |
   | 대기 단계의 사람 | 캠페인이 아직 실행되고 있다면, 캠페인에 남아 있는 사람들의 수입니다 *대기* 단계. |
   | 플로우에서 제거됨 | 캠페인에서 제거된 사람 수입니다. |
   | 마지막 활동 | 개인이 캠페인에 의해 처리된 마지막 시간입니다. |

   보고서의 캠페인 유형:

   | 유형 | 설명 |
   |---|---|
   | 일괄 처리 | 에서 캠페인을 묶습니다. **마케팅 활동** 탭. |
   | 작업 실행 | 개인 작업은 **데이터베이스**. |
   | 트리거됨(활성) | 에서 현재 트리거된 캠페인 **마케팅 활동** 탭. |
   | 트리거됨(비활성) | 에서 비활성 트리거된 캠페인 **마케팅 활동** 탭. |
