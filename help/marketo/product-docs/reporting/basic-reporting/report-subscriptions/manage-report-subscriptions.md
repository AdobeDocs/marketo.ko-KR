# 보고서 구독 관리 {#manage-report-subscriptions}

보고서 구독을 구성하고 삭제합니다.

1. 로 이동 **분석** 영역입니다.

   ![](assets/image2014-9-16-10-3a35-3a25.png)

1. 다음을 클릭합니다. **구독** 탭.

   ![](assets/image2014-9-16-10-3a35-3a32.png)

   이 페이지에는 계정의 모든 보고서에 대한 구독이 다음 기준으로 그룹화되어 표시됩니다. [보고서 유형](/help/marketo/product-docs/reporting/basic-reporting/report-types/report-type-overview.md). 여기에는 다음이 포함됩니다 [기본 보고서에 대한 구독](/help/marketo/product-docs/reporting/basic-reporting/report-subscriptions/subscribe-to-a-basic-report.md) 수익 주기 탐색기 보고서.

   >[!TIP]
   >
   >에서 개별 보고서에 대한 구독을 관리할 수도 있습니다. **마케팅 활동**. 보고서를 선택하고 **구독** 탭.

   보고서를 이메일로 보내는 빈도를 변경하려면 빈도 필드를 클릭하고 드롭다운 메뉴에서 새 옵션을 선택합니다.

   ![](assets/image2014-9-16-10-3a36-3a4.png)

1. 구독에서 이메일 주소를 변경하려면 **이메일 수신자** 이메일 주소를 필드에 추가하고 편집합니다.

   ![](assets/image2014-9-16-10-3a36-3a11.png)

   >[!TIP]
   >
   >* 이메일 주소 사이에 쉼표를 사용합니다.
   >* 편집 내용을 저장하려면 영역을 클릭합니다 _외부_ 구독 목록입니다.

   다음을 수행할 수도 있습니다.

   * 다음을 클릭합니다. **보기** 단추를 클릭하여 보고서를 엽니다.
   * 선택 취소 **활성** 구독을 비활성화하는 확인란입니다.
   * 클릭하여 편집 **요약** 이메일에 표시되는 미리보기 행 수를 변경할 필드입니다.
   * 선택 취소 **Excel** 스프레드시트 첨부 파일 없이 보고서 요약을 보내는 확인란
   * 다음을 클릭합니다. **보내기** 보고서 이메일을 즉시 전송하는 단추입니다.

1. 구독을 모두 삭제하려면 행을 선택하고 을(를) 클릭합니다. **구독 삭제**.

   ![](assets/image2014-9-16-10-3a36-3a38.png)

1. 구독 삭제 의사를 확인합니다.

   ![](assets/image2014-9-16-10-3a36-3a43.png)
