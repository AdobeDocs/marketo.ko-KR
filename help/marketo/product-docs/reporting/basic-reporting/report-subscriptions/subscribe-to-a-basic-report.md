# 기본 보고서 구독 {#subscribe-to-a-basic-report}

기본 보고서의 자동 업데이트를 받거나 기본 보고서를 공유하려면 모든 이메일 주소를 기존 보고서에 구독할 수 있습니다.

>[!NOTE]
>
>매출 주기 탐색기 보고서 가입의 경우 다음을 참조하십시오 [매출 탐색기 보고서 구독](/help/marketo/product-docs/reporting/revenue-cycle-analytics/revenue-explorer/subscribe-to-a-revenue-explorer-report.md).

1. 로 이동합니다. **마케팅 활동** 영역.

   ![](assets/image2014-9-16-10-3a31-3a54.png)

1. 탐색 트리에서 보고서를 선택하고 **구독** 탭.

   ![](assets/image2014-9-16-10-3a32-3a1.png)

   >[!NOTE]
   >
   >또한 **Analytics** 탭.

1. 클릭 **새 보고서 구독**.

   ![](assets/image2014-9-16-10-3a32-3a24.png)

1. 이메일 주소를 입력하고 보고서 이메일의 빈도를 설정합니다.

   ![](assets/image2014-9-16-10-3a32-3a31.png)

   >[!NOTE]
   >
   >누구나 수신한 이메일에서 보고서 구독을 취소할 수 있습니다.

   됐습니다. 받은 편지함을 확인하십시오!

   ![](assets/image2014-9-16-10-3a32-3a49.png)
