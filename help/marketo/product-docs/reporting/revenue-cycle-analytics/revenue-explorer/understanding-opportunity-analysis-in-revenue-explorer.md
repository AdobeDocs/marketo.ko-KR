# 매출 탐색기에서 기회 분석 이해 {#understanding-opportunity-analysis-in-revenue-explorer}

매출 탐색기의 기회 분석을 사용하면 보다 심층적으로 기회를 조사할 수 있습니다. 리드 소스, 업계 또는 지리적 위치와 같은 리드/회사 속성을 기반으로 데이터를 슬라이스하여 도식합니다. 이름, 단계 또는 확률을 기반으로 기회 창출 및 성사 분석 파이프라인에 대한 마케팅 기여도를 확인하십시오.

## 예제 분석 {#example-analysis}

다음은 기회 분석 영역에서 만들 수 있는 몇 가지 보고서입니다.

1. 마케팅이 생성된 기회들에 미치는 영향

   귀사의 파이프라인의 몇 퍼센트가 마케팅 프로그램의 영향을 받았습니까? 이 보고서는 정답을 준다. 다음 파이 차트는 마케팅 프로그램에서 얻은 모든 기회 수와 기회 금액의 백분율을 보여줍니다.

   ![](assets/image2015-7-21-16-3a21-3a4.png)

1. 마케팅이 영업 기회에 미치는 영향
1. 이 보고서는 마케팅에 의해 취득되고 영향을 받은 매출을 숫자와 금액별로 보여줍니다.

   ![](assets/image2015-7-21-16-3a41-3a55.png)

1. 리드 소스로 마감된 기회
1. 이 보고서는 리드 소스로 마감된 모든 기회를 분석하고, 효과가 있고 효과가 없는 소스에 대한 명확한 개요를 제공합니다.

   ![](assets/image2015-7-21-10-3a34-3a50.png)

1. 소스별 종료 시간

   이 보고서는 기회를 종료하기 위한 평균 일 수와 리드 소스 간의 관계를 보여줍니다.

   ![](assets/image2015-7-21-10-3a35-3a3.png)

1. 기회 및 단계 열기

   이 보고서는 각 매출 주기 단계에서 열려 있는 기회 수를 보여줍니다.

   ![](assets/image2015-7-21-10-3a35-3a32.png)

1. 업계별 연간 기회 수

   이 보고서는 &quot;우리는 매년 특정 산업에서 점점 더 혹은 더 적은 기회를 얻고 있습니까?&quot;라는 질문에 답한다.

   ![](assets/image2015-7-21-10-3a35-3a45.png)

## 기회 분석 Dimension 및 조치 {#opportunity-analysis-dimensions-and-measures}

기회 분석을 통해 모든 리드, 회사 및 기회 관련 차원 및 기회 관련 조치에 액세스할 수 있습니다. 이러한 기회 분석 차원 및 측정을 사용하여 보고서의 특정 질문에 답변합니다.

1. 회사 속성

   | Dimension | 설명 |
   |---|---|
   | 연간 매출 | 회사의 연간 매출 |
   | 구/군/시 | 회사가 위치한 도시 |
   | 국가 | 회사가 위치한 주 |
   | 업계 | 그 회사의 산업 |
   | 회사 이름 | 회사 이름 |
   | 직원 수 | 회사의 직원 수 |
   | 우편 번호 | 회사 우편 번호 |
   | SIC 코드 | 회사 SIC 코드 |
   | 주 | 회사가 위치한 주 |

1. 리드 속성

   | Dimension | 설명 |
   |---|---|
   | 차단 목록에 추가된 | 리드 차단 목록에 추가된 |
   | 기회로 전환 | 리드가 기회로 전환됩니다. |
   | 이메일 잘못됨 | 리드에 유효한 이메일 주소가 있는지 여부 |
   | 마케팅 일시 중단 | 마케팅 이메일에서 리드가 일시 중단됩니까? |
   | 이메일 주소 | 리드의 이메일 주소 |
   | 직함 | 리드의 직함 |
   | 전체 이름 | 리드의 전체 이름 |
   | 원본 소스 유형 | 리드의 원래 소스 유형 |
   | 소스 유형 등록 | 리드의 등록된 소스 유형 |
   | 리드 소유자 이메일 주소 | 리드 소유자의 이메일 주소 |
   | 리드 소유자 직책 | 리드 소유자의 직책 |
   | 리드 소유자 이름 | 리드 소유자 이름 |
   | 리드 소스 | 리드 소스 |
   | 리드 상태 | 리드 상태 |

1. 리드 생성 일정

   | Dimension | 설명 |
   |---|---|
   | 리드 생성 연도 | 리드가 생성된 연도 |
   | 리드 생성 분기 | 리드가 생성된 분기 |
   | 월에 생성된 리드 | 리드가 생성된 달 |
   | 리드 생성 주 | 리드가 생성된 주 |
   | 리드 생성 날짜 | 리드를 만든 날짜 |

1. 기회 속성

   | Dimension | 설명 |
   |---|---|
   | 영업 기회 마감 | 기회가 끝났나요 |
   | 기회 예측 범주 | 기회 예측 범주 |
   | 영업 기회 이름 | 기회 이름 |
   | 기회 단계 | 기회 단계 |
   | 기회 유형 | 기회 유형 |
   | 기회 원 | 이 기회가 끝나고 이겼습니까 |
   | 마케팅 영향 기회 | 이 플래그는 모든 마케팅 프로그램에서 리드/연락처가 획득 또는 성공했는지 여부를 나타냅니다. 기간 비용이 정의된 프로그램만 고려됩니다. |

1. 영업 기회 마감 일정

   | Dimension | 설명 |
   |---|---|
   | 영업 기회 마감 연도 | 기회가 끝나는 해 |
   | 영업 기회 마감 분기 | 기회가 종료되는 분기 |
   | 영업 기회 마감 달 | 기회가 종료되는 달 |
   | 영업 기회 마감 주 | 기회가 끝나는 주 |
   | 영업 기회 마감 날짜 | 기회가 종료된 날짜 |

1. Opportunity 생성 일정

   | Dimension | 설명 |
   |---|---|
   | Opportunity Created Year | 기회가 만들어진 연도 |
   | 영업 기회 창출 분기 | 기회가 생성된 분기 |
   | Opportunity Created Month | 기회가 만들어진 달 |
   | Opportunity Created Week | 기회가 만들어진 주 |
   | 기회 생성 날짜 | 기회가 생성된 날짜 |

1. 조치

   | 측정 | 설명 |
   |---|---|
   | 영업 기회 종료 평균 일 수 | 영업 기회를 마감할 평균 일 수 |
   | 영업 기회 종료 평균 일 수(손실) | 잃어버린 기회까지의 평균 일 수 |
   | 영업 기회 종료일 평균(원) | 획득 기회의 평균 일 수 |
   | 모든 기회 수 | 모든 총 기회 수 |
   | 기회 수(마감) | 종료된 총 기회 수(원 또는 손실) |
   | 기회 수(손실) | 손실된 총 기회 수 |
   | 기회 수(개설) | 아직 열려 있는 총 기회 수 |
   | 기회 수(원) | 획득한 총 기회 수 |
   | 기회 금액 | 총 기회 금액 두 개 이상의 리드가 기회와 연관된 경우 할당 금액은 리드 점수를 기반으로 합니다. |
   | 기회 금액(손실) | 손실된 기회에 대한 총 금액 두 개 이상의 리드가 기회와 연관된 경우 할당 금액은 리드 점수를 기반으로 합니다. |
   | 기회 금액(개설) | 오픈 기회 총 금액 두 개 이상의 리드가 기회와 연관된 경우 할당 금액은 리드 점수를 기반으로 합니다. |
   | 기회 금액(원) | 획득 기회의 총 금액입니다. 두 개 이상의 리드가 기회와 연관된 경우 할당 금액은 리드 점수를 기반으로 합니다. |
