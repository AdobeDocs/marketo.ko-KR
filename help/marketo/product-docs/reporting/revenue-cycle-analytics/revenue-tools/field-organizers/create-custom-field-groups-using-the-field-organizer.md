# 필드 구성자를 사용하여 사용자 지정 필드 그룹 생성 {#create-custom-field-groups-using-the-field-organizer}

매출 주기 탐색기의 모델 성과 분석(리드) 영역에서 보고를 위해 사용자 지정 필드 그룹을 활성화하려면 먼저 Marketo 리드 관리의 필드 구성자를 통해 표준 또는 사용자 지정 필드를 보고 그룹으로 분류해야 합니다. 이것은 리드 및 회사 속성에만 적용됩니다.
새 필드 구성자 대화 상자의 필드 드롭다운에서 표준 또는 사용자 지정 필드를 선택하면 시스템은 그룹화할 필드와 연관된 Marketo Lead Management 데이터 유형을 필드 구성자에서 사용 가능한 세 명의 편집기 중 하나로 매핑합니다. 문자열, 정수 또는 날짜

| Marketo 리드 관리 데이터 유형 | 필드 구성자 편집기 데이터 유형 |
|---|---|
| 문자열 | 문자열 |
| 이메일 | 문자열 |
| 정수 | 정수 |
| 텍스트 | 문자열 |
| URL | 문자열 |
| 참조 | 지원되지 않음 |
| 통화 | 정수 |
| DateTime | 날짜 |
| 부울 | 지원되지 않음 |
| 휴대폰 | 문자열 |
| 날짜 | 날짜 |
| 부동 | 정수 |
| 계산된 지표 | 지원되지 않음 |

다음 세 섹션에서는 문자열, 정수 또는 날짜 유형에 대한 사용자 지정 필드 그룹을 만드는 방법을 설명합니다.

## 사용자 지정 필드 그룹 만들기 - 문자열 편집기 {#create-custom-field-group-string-editor}

1. 클릭 **리드 데이터베이스**.

   ![](assets/one.png)

1. 클릭 **새로 만들기**, 을(를) 선택하고 을(를) 선택합니다. **새 필드 구성자**.

   ![](assets/two.png)

1. 클릭 **필드** 및 문자열 편집기에 매핑되는 데이터 유형을 사용하는 표준 또는 사용자 지정 필드를 선택합니다(이전 섹션의 표 참조). 국가는 여기에서 사용됩니다.

   ![](assets/three.png)

1. Click **Create**.

   ![](assets/four.png)

   새 사용자 지정 그룹은 필드 이름 > 필드 이름 그룹으로 표시되는 리드 데이터베이스 트리에 표시됩니다(예: 국가 > 국가 그룹).

   ![](assets/4.5.png)

1. 연필 아이콘을 클릭하여 이름을 사용자 지정합니다. 예를 들어 &quot;국가 그룹&quot;의 이름을 &quot;대륙&quot;으로 바꿀 수 있습니다. 원하는 새 이름을 입력하고 상자에서 클릭하여 자동으로 저장합니다.

   ![](assets/five.png)

1. 기본적으로 모든 데이터 값은 &quot;기타&quot; 하위 그룹에 배치됩니다. 데이터 값을 분류하려면 **그룹 추가** 하위 그룹을 만들고 이름을 지정합니다.

   >[!NOTE]
   >
   >최대 10개의 하위 그룹을 추가하여 데이터 값을 분류할 수 있습니다. 생성된 각 하위 그룹에는 ID 번호가 지정됩니다.

   이 예에서는 대부분의 대륙에 대해 그룹이 생성되었습니다.

   ![](assets/six.png)

   >[!NOTE]
   >
   >하위 그룹을 삭제하려면 하위 그룹 이름 옆에 있는 빨간색 X를 클릭하면 됩니다. 그룹에 데이터 값이 있으면 데이터 값이 기본 그룹인 기타(Other)로 이동합니다.

1. 캔버스에서 데이터 값을 강조 표시하고 데이터 값을 적절한 하위 그룹으로 끌어다 놓습니다.

   ![](assets/seven.png)

   >[!NOTE]
   >
   >하위 그룹에서 데이터 값을 제거하려면 데이터 값을 기본 그룹인 기타(Other)에 재할당합니다.

1. 캔버스 바로 위의 왼쪽 위 모서리에 있는 필터 옵션을 사용하여 하나 이상의 하위 그룹에서 데이터 값을 선택하고 볼 수 있습니다. 필터 선택을 기반으로 하는 데이터 값이 캔버스에 표시됩니다.

   ![](assets/eight.png)

   >[!NOTE]
   >
   >그룹이 정의되면 Marketo Lead Management의 Revenue Cycle Analytics 탭을 통해 Model Performance Analysis(Leads)에서 보고할 사용자 지정 필드 그룹을 활성화할 수 있습니다.

## 사용자 지정 필드 그룹 만들기 - 정수 편집기 {#create-custom-field-group-integer-editor}

1. 클릭 **리드 데이터베이스**.

   ![](assets/one.png)

1. 클릭 **새로 만들기**, 을(를) 선택하고 을(를) 선택합니다. **새 필드 구성자**.

   ![](assets/two.png)

1. 클릭 **필드** 및 문자열 편집기에 매핑되는 데이터 유형을 사용하는 표준 또는 사용자 지정 필드를 선택합니다(이전 섹션의 표 참조). 연간 매출은 여기에서 사용됩니다.

   ![](assets/nine.png)

1. Click **Create**.

   ![](assets/9.5.png)

   새 사용자 지정 그룹은 필드 이름 > 필드 이름 그룹으로 표시되는 리드 데이터베이스 트리에 표시됩니다(예: 연간 매출 > 연간 매출 그룹).

   ![](assets/9.6.png)

1. 정수 편집기 위에 있는 기본 사용자 지정 그룹 이름을 클릭하여 이름을 사용자 지정합니다. 예를 들어 &quot;연간 매출 그룹&quot;의 이름을 &quot;크기별 연간 수입&quot;으로 바꿀 수 있습니다. 클릭 **저장**.

   ![](assets/eleven.png)

   정수 편집기를 사용하면 여러 하위 그룹을 만들어 각 하위 그룹을 크기별로 정의할 수 있습니다. 이 예에서는 Small, Medium 및 Enterprise 기업을 위해 세 개의 그룹이 작성됩니다.

1. 첫 번째 그룹을 추가하려면 **그룹 이름** 필드(예: 작음)에 최대 값을 입력합니다. **그룹 범위** 필드(예: 200000). 클릭 **그룹 추가**.

   ![](assets/twelve.png)

   방금 입력한 그룹 아래에 빈 그룹 항목이 표시됩니다. 아래 예는 Small, Medium 및 Enterprise Business 항목을 보여줍니다.

   >[!NOTE]
   >
   >최대 10개의 하위 그룹을 추가하여 데이터 값을 분류할 수 있습니다. 각 그룹 범위 항목은 이전 항목을 기반으로 빌드됩니다. 마지막으로 만든 사용자 지정 하위 그룹에 대해 마지막 그룹 범위 항목을 비워 두면 최대 데이터 값이 설정되지 않습니다.

1. 요약 탭을 클릭하여 설정을 저장하고 검토합니다.

   ![](assets/thirteen.png)

   >[!NOTE]
   >
   >하위 그룹을 삭제하려면 하위 그룹 이름 옆에 있는 빨간색 X를 클릭합니다.

1. 요약 페이지에서 설정을 검토합니다.

   ![](assets/13.5.png)

   >[!NOTE]
   >
   >그룹이 정의되면 Marketo Lead Management의 Revenue Cycle Analytics 탭을 통해 Model Performance Analysis(Leads)에서 보고할 사용자 지정 필드 그룹을 활성화할 수 있습니다.

## 사용자 지정 필드 그룹 만들기 - 날짜 편집기 {#create-custom-field-group-date-editor}

1. 클릭 **리드 데이터베이스**.

   ![](assets/one.png)

1. 클릭 **새로 만들기**, 을(를) 선택하고 을(를) 선택합니다. **새 필드 구성자**.

   ![](assets/two.png)

1. 클릭 **필드** 및 문자열 편집기에 매핑되는 데이터 유형을 사용하는 표준 또는 사용자 지정 필드를 선택합니다(이전 섹션의 표 참조). 획득 날짜는 여기에서 사용됩니다.

   ![](assets/fourteen.png)

1. Click **Create**.

   ![](assets/14.5.png)

   새 사용자 지정 그룹은 필드 이름 > 필드 이름 그룹으로 표시되는 리드 데이터베이스 트리에 표시됩니다(예: 획득 날짜 > 획득 날짜 그룹).

   ![](assets/14.6.png)

1. 날짜 편집기 위에 있는 기본 사용자 지정 그룹 이름을 클릭하여 이름을 사용자 지정합니다. 예를 들어 &quot;획득 날짜 그룹&quot;의 이름을 &quot;획득 날짜 카테고리&quot;로 바꿀 수 있습니다. 클릭 **저장**.

   ![](assets/fifteen.png)

   날짜 편집기를 사용하면 여러 개의 하위 그룹을 만들고 날짜별로 각 하위 그룹을 정의할 수 있습니다. 이 예에서는 세 개의 그룹이 만들어집니다. Q1-15 리드, Q2-15 리드, Q3-15 리드

1. 첫 번째 그룹을 추가하려면 **그룹 이름** 필드(예: Q1-15 Lead) 를 입력하고 Date 필드에 Lead가 Acquisition On 또는 Before에 취득된 일자를 입력합니다(예: 1-15분기 마지막 날 3/31/2015). 클릭 **그룹 추가**.

   ![](assets/sixteen.png)

   >[!NOTE]
   >
   >최대 10개의 하위 그룹을 추가하여 데이터 값을 분류할 수 있습니다. 각 그룹 범위 항목은 이전 항목을 기반으로 빌드됩니다. 마지막으로 만든 사용자 지정 하위 그룹에 대해 마지막 그룹 범위 항목을 비워 두면 종료 날짜 값이 설정되지 않습니다.

   아래 예제는 2015년 1사분기 리드에 대한 항목을 Q3까지 보여 줍니다.

   ![](assets/16.5.png)

   그게 다야! 잘했어요
