# 프로그램 정보를 표시하는 이메일 분석 보고서 작성 {#build-an-email-analysis-report-that-shows-program-information}

다음 단계에 따라 프로그램 채널별로 그룹화된 이메일 정보를 표시하는 이메일 분석 보고서를 만듭니다.

>[!NOTE]
>
>**가용성**
>
>모든 고객이 이 기능을 구입하지는 않았습니다. 자세한 내용은 영업 담당자에게 문의하십시오.

1. 매출 **탐색기를 시작합니다**.

   ![](assets/image2014-9-17-19-3a42-3a26.png)

1. 새 보고서 **를 클릭합니다**.

   ![](assets/image2014-9-17-19-3a42-3a32.png)

1. *** 이메일 분석* 영역을 선택하고 **확인을 클릭합니다**.

   ![](assets/image2014-9-17-19-3a43-3a20.png)

1. 전송( **주)** 노란색 점을 찾아 마우스 오른쪽 단추로 클릭합니다. 필터 **를 클릭합니다**.

   >[!NOTE]
   >
   >이렇게 하면 보고서의 기간이 좁아집니다.

   ![](assets/image2014-9-17-19-3a43-3a49.png)

1. [ **현재 보낸 주** ]를 선택하고 [ **확인]을 클릭합니다**.

   ![](assets/image2014-9-17-19-3a43-3a59.png)

1. ** 프로그램 채널*** 노란색 점을 찾아 두 번 클릭합니다.

   ![](assets/image2014-9-17-19-3a44-3a14.png)

1. 이메일 이름 **노란색 점을 찾아** 두 번 클릭합니다.

   ![](assets/image2014-9-17-19-3a44-3a34.png)

1. 전송됨, **열림**&#x200B;및 **클릭한** 파란색 점 **을** 찾아 두 번클릭합니다.

   ![](assets/image2014-9-17-19-3a44-3a41.png)

   잘했어! 다음과 같은 보고서가 있어야 합니다.

   ![](assets/image2014-9-17-19-3a45-3a1.png)

   >[!NOTE]
   >
   >**딥 다이브**
