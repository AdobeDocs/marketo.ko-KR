# 리드를 나열하는 프로그램 멤버십 분석 보고서 작성 {#build-a-program-membership-analysis-report-that-lists-leads}

모든 프로그램 멤버 이름을 보여 주는 프로그램 멤버 분석 보고서를 만들려면 다음 단계를 따르십시오.

>[!AVAILABILITY]
>
>모든 고객이 이 기능을 구입한 것은 아닙니다. 자세한 내용은 영업 담당자에게 문의하십시오.

1. Revenue Explorer를 시작합니다.

   ![](assets/one.png)

1. 클릭 **새로 만들기** 그런 다음 **보고서**.

   ![](assets/two.png)

1. 선택 **프로그램 멤버십 분석** 을(를) 클릭합니다. **확인**.

   ![](assets/three.png)

1. 프로그램 이름 노란색 점을 찾아 두 번 클릭합니다.

   ![](assets/four.png)

1. 멤버 파란색 점을 찾아 두 번 클릭합니다.

   ![](assets/five.png)

1. 성공 (합계) 파란색 점을 찾아 두 번 클릭합니다.

   ![](assets/six.png)

얼마나 쉬웠는지 알아?

![](assets/seven.png)

다음과 같은 리드 속성을 빠르게 추가할 수 있습니다. _전체 이름_ 구성원이 누구인지 확인하기 위해

>[!TIP]
>
>선택할 많은 리드/회사 특성이 있습니다. 확인해!

1. 을(를) 찾아 두 번 클릭합니다. **전체 이름** 노란색 점.

   ![](assets/eight.png)

여기 있어요!

![](assets/nine.png)
