# 명명된 계정 만들기 {#create-a-named-account}

다음 단계에 따라 명명된 계정을 수동으로 만듭니다.

1. 지정된 계정에서 **새로 만들기** 드롭다운 및 선택 **명명된 계정 만들기**.

   ![](assets/two-1.png)

1. 원하는 필드를 작성하고 을(를) 클릭합니다 **만들기**.

   ![](assets/three-1.png)

   >[!TIP]
   >
   >지정된 계정을 직접 클릭하여 해당 대시보드를 확인합니다.
