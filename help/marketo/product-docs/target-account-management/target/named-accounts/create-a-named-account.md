# 명명된 계정 만들기 {#create-a-named-account}

명명된 계정을 수동으로 만들려면 다음 단계를 따르십시오.

1. 명명된 계정에서 **신규** 드롭다운 및 선택 **명명된 계정 만들기**.

   ![](assets/two-1.png)

1. 원하는 필드를 입력하고 클릭 **만들기**.

   ![](assets/three-1.png)

   >[!TIP]
   >
   >명명된 계정을 직접 클릭하여 대시보드를 봅니다.
