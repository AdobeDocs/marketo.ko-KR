# RCA의 명명된 계정 Dimension {#named-account-dimension-in-rca}

Revenue Cycle Analytics에서 TAM별 명명 계정 차원을 사용하여 수익 기반 보고서를 작성합니다.

>[!NOTE]
>
>**Dimension** - 측정값의 다른 보기를 제공하는 속성(노란색 점으로 표시됨)입니다.

![](assets/one-2.png)

>[!NOTE]
>
>RCA의 명명 계정 차원은 타깃팅된 계정의 최종적인 영향을 측정하는 데 사용할 수 있습니다(예: 매출 원, 파이프라인 생성 또는 판매 주기 가속화). 이 차원을 사용하여 지정된 계정에 대해 잘 수행되고 제대로 수행되지 않은 프로그램을 식별할 수도 있습니다.

다음 보고서는 명명된 계정 차원에 액세스할 수 있습니다.

* 이메일 분석
* 리드 분석
* 기회 분석
* 프로그램 멤버십 분석

>
