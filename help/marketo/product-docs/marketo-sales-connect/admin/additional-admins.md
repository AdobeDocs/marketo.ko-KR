# 추가 관리자 {#additional-admins}

팀원을 변경하거나 제거하려면 설정 페이지로 이동하여 왼쪽 탐색 패널에서 **팀 관리****를 클릭합니다.
계정 관리자로서 여기에서 팀의 사용자를 관리할 수 있습니다.
