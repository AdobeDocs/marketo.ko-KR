# ID 추가 {#add-identity}

보낼 이메일 주소가 여러 개인 경우 ID를 추가합니다.

1. 설정 아이콘을 클릭합니다.

1. 클릭 **전자 메일 설정**.

1. 클릭 **ID 추가**.

1. 정보를 입력합니다(전자 메일 및 이름은 필수 필드). **만들기** 완료 시.

1. 받은 확인 이메일에서 링크를 클릭합니다.

   Sales Connect에서 이메일을 보낼 때 ID 간에 전환할 수 있습니다.

   >[!NOTE]
   >
   >설정할 수 있는 이메일 ID 수에는 제한이 없습니다.
