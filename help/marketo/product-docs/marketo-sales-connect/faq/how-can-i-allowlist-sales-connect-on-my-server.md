# 내 서버에서 Sales Connect를 허용 목록에 추가하다 세그먼트화하려면 어떻게 해야 합니까? {#how-can-i-allowlist-sales-connect-on-my-server}

안타깝게도 하나의 정적 IP 주소를 제공할 방법이 없습니다. Sales Connect는 클라우드에 배포되며 200개 이상의 서버를 사용하여 전자 메일 및 사용자 요청을 처리합니다. 그리고 아주 넓은 범위를 제공하는 것은 더 이상 여러분에게 안전하지 않을 것입니다.
