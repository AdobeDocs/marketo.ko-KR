# 템플릿 즐겨찾기 {#favorite-a-template}

모든 범주의 템플릿을 즐겨찾기 목록에 추가할 수 있습니다. 이렇게 하면 가장 많이 사용하는 템플릿에 빠르게 액세스할 수 있도록 목록의 맨 위에 새 카테고리가 자동으로 생성됩니다.

즐겨찾기에 추가하려면 템플릿 옆에 있는 별 아이콘을 클릭합니다. 템플릿 목록 또는 편집기 창에서 이 작업을 수행할 수 있습니다.
