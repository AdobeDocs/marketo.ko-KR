# 템플릿 보기 사용자 지정 {#customize-template-view}

**보기** 드롭다운에서 모든 템플릿, 템플릿, 공유 템플릿 또는 사용되지 않은 템플릿(지난 90일 동안 사용되지 않은 템플릿)을 표시하도록 선택할 수 있습니다.

또한 **보기.** 오른쪽에 있는 설정 아이콘을 클릭하여 템플릿 보기에서 열을 추가/제거할 수 있습니다.
