# ToutApp 브라우저 확장 기능 설치 방법 {#how-to-install-the-toutapp-browser-extension}

1. 클릭 **설정** 다음에 있음 [웹 애플리케이션](https://toutapp.com/next#settings).

1. 내 계정에서 **이메일 설정**.

1. 클릭 **플러그인**.

1. Gmail 및 Google 앱에서 **설치**.
