# ToutApp 브라우저 확장 설치 방법 {#how-to-install-the-toutapp-browser-extension}

1. 클릭 **설정** on [웹 애플리케이션](https://toutapp.com/next#settings).

1. 내 계정에서 **전자 메일 설정**.

1. 클릭 **Plugins**.

1. Gmail 및 Google 앱에서 을 클릭합니다. **설치**.
