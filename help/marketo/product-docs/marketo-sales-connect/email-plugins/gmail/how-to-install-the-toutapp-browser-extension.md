# ToutApp 브라우저 확장 기능 설치 방법 {#how-to-install-the-toutapp-browser-extension}

1. [웹 응용 프로그램](https://toutapp.com/next#settings)에서 **설정**&#x200B;을 클릭합니다.

1. 내 계정에서 **전자 메일 설정**&#x200B;을 클릭합니다.

1. **플러그인**&#x200B;을 클릭합니다.

1. Gmail 및 Google 앱에서 **설치**&#x200B;를 클릭합니다.
