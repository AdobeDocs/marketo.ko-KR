# 달력 기능 {#calendar-functionality}

을 눌러 사용 가능한 경우 수신자에게 알립니다 **달력** 버튼을 클릭합니다. Google 달력에서 사용자가 자유롭게 사용할 수 있는 특정 날짜 및 시간을 추가할 수 있도록 해주며 수신자는 사용자에게 적합한 시간으로 답글을 제공할 수 있습니다.
