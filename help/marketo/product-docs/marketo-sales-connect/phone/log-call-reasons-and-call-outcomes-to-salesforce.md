# Salesforce에 통화 이유 및 호출 결과 기록 {#log-call-reasons-and-call-outcomes-to-salesforce}

보고 또는 가시성을 위해 Salesforce에 호출 결과를 기록하고 호출 이유를 기록하려는 경우 각각에 대해 사용자 지정 활동 필드를 만들 수 있습니다. 각 필드는 특정 API 이름을 사용해야 합니다.

* 호출 결과 API 이름: mktosales_call_result
* 호출 이유 API 이름: mktosales_call_reason

이러한 필드를 활용하려면 먼저 필드를 사용자 지정 활동 필드로 만들어야 합니다. 사용자가 볼 수 있도록 하려면 작업 개체 페이지 레이아웃에 추가해야 합니다.

## Salesforce Classic에서 사용자 지정 활동 필드 만들기  {#create-custom-activity-field-in-salesforce-classic}

1. Salesforce에서 설정으로 이동합니다.

PICC

1. 빠른 찾기 상자에 &quot;활동&quot;을 입력합니다.

PICC

1. 클릭 **활동 사용자 지정 필드**.

PICC

1. 클릭 **새로 만들기**.

PICC

## Salesforce Lightning에서 사용자 지정 활동 필드 만들기 {#create-custom-activity-field-in-salesforce-lightning}

1. Salesforce에서 오른쪽 위의 톱니바퀴 아이콘을 클릭합니다.

PICC

1. 클릭 **설정**.

PICC

1. 클릭 **개체 관리자**.

PICC

1. 빠른 찾기 상자에 활동을 입력하고 활동 레이블을 클릭하여 객체의 설정을 엽니다.

PICC

1. 왼쪽에서 **필드 및 관계**.

PICC

1. 클릭 **새로 만들기**.

PICC
