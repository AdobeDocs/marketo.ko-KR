# 캠페인이 종료됨 {#campaign-has-ended}

게재 실패 폴더에 &quot;캠페인이 종료되었습니다&quot;라는 이메일이 표시되는 경우 이 특정 이메일이 캠페인에서 예약된 이메일임을 의미합니다.

## 실패한 가능한 원인 {#possible-reasons-it-failed}

* 반송되기 전 이메일
* 연락처가 회신하여 캠페인을 종료했습니다
* 캠페인이 수동으로 종료됨
