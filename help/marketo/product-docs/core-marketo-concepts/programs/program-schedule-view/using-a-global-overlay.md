# 글로벌 오버레이 사용 {#using-a-global-overlay}

프로그램 예약 보기의 전역 오버레이를 사용하면 다른 예약된 자산과 관련하여 프로그램을 볼 수 있습니다.

>[!PREREQUISITES]
>
>다음을 수행해야 합니다. [마케팅 달력 라이선스](/help/marketo/product-docs/core-marketo-concepts/marketing-calendar/understanding-the-calendar/issue-revoke-a-marketing-calendar-license.md) 이 기능을 활용하려면 다음을 수행하십시오.

## 글로벌 오버레이 사용 {#use-the-global-overlay}

1. 프로그램을 선택합니다.

   ![](assets/image2014-9-24-10-16-4.png)

1. 선택 **오버레이** 오른쪽 아래 모서리에 있습니다.

   ![](assets/image2014-9-24-10-3a16-3a9.png)

1. 솔리드 블록은 해당 날짜의 항목을 나타냅니다. 자세한 내용을 보려면 을(를) 클릭합니다.

   ![](assets/image2014-9-24-10-3a16-3a14.png)

   오버레이 항목의 항목 세부 정보는 읽기 전용입니다. 변경하려면 상위 프로그램을 클릭하십시오.

   ![](assets/image2014-9-24-10-3a16-3a19.png)

## 저장된 필터를 오버레이로 사용 {#use-a-saved-filter-as-an-overlay}

만약 [마케팅 달력에 필터 저장](/help/marketo/product-docs/core-marketo-concepts/marketing-calendar/working-with-the-calendar/saving-a-filter-definition-in-the-marketing-calendar.md)를 입력하면 프로그램 예약 보기에서 오버레이로 사용할 수 있습니다.

1. 을(를) 클릭합니다. **오버레이** 드롭다운을 클릭하고 필터 정의를 선택합니다.

   ![](assets/image2014-9-24-10-3a16-3a26.png)

   그렇게 쉬워! 이제 저장하고 선택한 필터에 의해 정의된 오버레이가 표시됩니다.

   ![](assets/image2014-9-24-10-3a16-3a31.png)
