# 프로그램 스케줄 보기에서 배치 Smart Campaign의 스케줄 조정 {#reschedule-a-batch-smart-campaign-in-the-program-schedule-view}

일괄 처리 스마트 캠페인 실행을 다시 예약하시겠습니까? 프로그램 예약 보기에서 나중에 끌어서 놓습니다.

1. 다시 예약하려는 스마트 캠페인을 선택합니다.

   ![](assets/image2015-5-19-12-3a8-3a28.png)

1. 캠페인을 실행할 새 날짜로 끌어다 놓습니다.

![](assets/image2015-5-19-12-3a12-3a1.png)

1. 좋아요! 스마트 캠페인 일정이 다시 조정되었습니다. 시간은 **날짜** 드롭다운 을 클릭합니다.

   ![](assets/image2015-5-19-12-3a15-3a38.png)
