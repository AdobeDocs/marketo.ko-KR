# 프로그램 복제 {#clone-a-program}

모든 항목을 수동으로 다시 작성하는 대신 전체 프로그램 및 모든 자산을 빠르고 쉽게 복제할 수 있습니다.

>[!PREREQUISITES]
>
>안으로 들어가기 전에 [복제된 자산의 손상 방지 우수 사례](https://nation.marketo.com/t5/Knowledgebase/Avoiding-Corruption-in-Cloned-Assets/ta-p/249729).

1. 복제할 프로그램을 찾아 선택한 다음 **프로그램 작업** click **복제**.

   ![](assets/image2014-9-5-14-3a31-3a49.png)

1. 프로그램을 복제할 위치를 선택합니다.

   >[!NOTE]
   >
   >프로그램을 [campaign](/help/marketo/product-docs/core-marketo-concepts/miscellaneous/create-new-campaign-folder.md) [폴더](/help/marketo/product-docs/core-marketo-concepts/miscellaneous/create-new-campaign-folder.md) 또는 [작업 공간](/help/marketo/product-docs/administration/workspaces-and-person-partitions/create-a-new-workspace.md). 기본 종속 자산(전자 메일, 코드 조각, 랜딩 페이지 템플릿 등)을 모두 공유해야 합니다. 복제 전 대상 작업 영역으로

   ![](assets/cloneto.png)

   >[!NOTE]
   >
   >위의 스크린샷에 이 NOTE가 표시됩니까? 즉, 목록에 1000명 이상의 사람을 사용하여 프로그램을 복제하면 목록 자체가 복제되지만 비어 있게 됩니다. 999명 이하의 사용자를 포함하는 목록으로 프로그램을 복제하면 해당 목록이 모든 해당 멤버와 함께 복제된 프로그램에 표시됩니다.

1. 이름을 입력합니다.

   ![](assets/cloneprogramname.png)

1. 프로그램을 복제할 폴더를 선택합니다.

   ![](assets/choosefolderclone.png)

1. 선택적 설명을 추가한 다음 **만들기**. 모든 자산이 준비된 새로운 프로그램을 얼마나 빨리 만들 수 있었는지 보십시오!

   ![](assets/createclone.png)

   >[!TIP]
   >
   >이 기술을 토큰과 함께 사용하여 새 프로그램을 신속하게 만들 수 있습니다.

   >[!CAUTION]
   >
   >기간 비용은 전달되지 않으므로 원래 세트에 한 개가 있는 경우 복제된 프로그램에 해당 내용을 추가해야 합니다.
