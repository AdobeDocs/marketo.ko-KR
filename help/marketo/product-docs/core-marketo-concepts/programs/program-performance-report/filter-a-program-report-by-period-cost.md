# 기간 원가로 프로그램 보고서 필터링 {#filter-a-program-report-by-period-cost}

특정 기간 원가 기간에 [프로그램 성과 보고서를](create-a-program-performance-report.md) 집중합니다.

1. 마케팅 **활동** (또는 **Analytics** )으로 **이동합니다**.

   ![](assets/login-marketing-activities-1.png)

1. 프로그램 성과 보고서를 선택합니다.

   ![](assets/image2014-9-23-16-3a22-3a52.png)

1. 설정 **탭을** 클릭하고 기간 비용 **위로 드래그합니다**.

   ![](assets/lm-86194-1.png)

1. 비용 **에** 대해 **** 시작 및 종료 날짜를 **입력하고 적용을**&#x200B;클릭합니다.

   ![](assets/lm-86194-2a-hands.png)

1. 이제 끝났어요! 보고서 **탭을** 클릭하여 지정된 기간 비용 기간 내에 있는 프로그램만 확인합니다.

   ![](assets/lm-86194-report-tab.png)
