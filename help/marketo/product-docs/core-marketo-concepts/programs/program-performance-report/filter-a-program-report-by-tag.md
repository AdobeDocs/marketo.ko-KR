# 태그로 프로그램 보고서 필터링 {#filter-a-program-report-by-tag}

집중하기 [프로그램 성과 보고서](/help/marketo/product-docs/core-marketo-concepts/programs/program-performance-report/create-a-program-performance-report.md) 특정 [태그](/help/marketo/product-docs/core-marketo-concepts/programs/working-with-programs/understanding-tags.md).

1. 이동 **마케팅** **활동** 또는 **Analytics**).

   ![](assets/login-marketing-activities.png)

1. 을(를) 선택합니다 **프로그램 성능** 보고서 세트에 대해 설명합니다.

   ![](assets/image2014-9-23-16-3a12-3a36.png)

1. 을(를) 클릭합니다. **설정** 탭을 선택하고 한 개 위로 드래그합니다. **태그** 필터 .

   ![](assets/prospects.jpg)

1. 보고서에 포함할 태그 값을 선택합니다.

   ![](assets/prospect1.jpg)

1. 클릭 **적용**.

   ![](assets/prospect2.jpg)

1. 완료! 을(를) 클릭합니다. **보고서** 탭을 클릭하여 _그냥_ 보고서에서 선택한 태그와 일치하는 프로그램입니다.

   ![](assets/image2014-9-23-16-3a14-3a42.png)
