# Excel로 스마트 캠페인 결과 내보내기 {#export-smart-campaign-results-to-excel}

Marketo 외부에서 사용할 스마트 캠페인 결과를 Excel로 내보냅니다.

1. 스마트 캠페인에서 을(를) 클릭합니다. **결과** 그리고 맨 아래에 내보내기 아이콘이 표시됩니다.

   ![](assets/exportexcel-hands.png)

   즉시 결과를 내보내고 사용할 준비가 되었습니다!

   >[!NOTE]
   >
   >내보내기는 20,000개의 행으로 제한됩니다. Excel 파일은 브라우저의 기본 다운로드 폴더에 저장됩니다.
