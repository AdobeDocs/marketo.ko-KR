# 필드에 데이터 추가 {#append-data-to-a-field}

데이터를 필드에 쉽게 추가할 수 있습니다. 방법은 다음과 같습니다.

>[!PREREQUISITES]
>
>[캠페인 만들기](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/create-a-new-smart-campaign.md){target="_blank"}

>[!NOTE]
>
>아래 단계는 [프로그램 구성원 데이터 변경](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/change-program-member-data.md){target="_blank"}에도 적용됩니다.

1. **[!UICONTROL 흐름]** 탭에서 **[!UICONTROL 데이터 값 변경]** 흐름 단계를 드래그합니다.

   ![](assets/append-data-to-a-field-1.png)

1. 데이터를 추가할 필드를 찾아 선택합니다.

   ![](assets/append-data-to-a-field-2.png)

1. 데이터를 추가할 동일한 필드에 대한 토큰을 찾아 선택합니다.

   ![](assets/append-data-to-a-field-3.png)

1. 이제 추가하려는 값을 필드에 이미 있는 값에 추가합니다.

   ![](assets/append-data-to-a-field-4.png)

다 됐습니다! 크리에이티브를 얻고 여러 토큰을 추가할 수 있습니다.
