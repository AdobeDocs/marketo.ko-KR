# 흐름 단계에서 선택 추가 사용 {#use-add-choice-in-a-flow-step}

>[!PREREQUISITES]
>
>* [스마트 캠페인에 흐름 단계 추가](add-a-flow-step-to-a-smart-campaign.md)

>



**선택** 추가를 사용하면 플로우 단계를 사용하여 세부 사항을 선택할 때 &quot;종속적&quot;으로 말할 수 있습니다.

1. 고급 캠페인의 **흐름** 탭 아래에서 흐름 단계를 추가한 다음 선택 **추가를 클릭합니다**.

   ![](assets/image2014-9-22-11-3a58-3a20.png)

1. 선택 조건을 선택합니다.

   ![](assets/image2014-9-22-11-3a58-3a50.png)

1. 선택 연산자를 선택하고 선택 값을 입력합니다. 기준이나 선택이 설정됩니다.

   ![](assets/image2014-9-22-11-3a58-3a54.png)

1. 선택할 플로우 단계 값을 입력합니다.

   ![](assets/image2014-9-22-11-3a58-3a57.png)

   >[!CAUTION]
   >
   >선택 흐름 단계의 조건에서는 토큰이 **작동하지 않습니다** .

1. 위의 단계를 반복하여 선택 항목을 여러 개 추가한 다음 기본값을 추가/조정합니다.

   ![](assets/image2014-9-22-11-3a58-3a59.png)

   >[!TIP]
   >
   >흐름 단계를 —Do Nothing—(아무 것도 안 함)으로 설정할 수 있습니다. 이 경우 해당 선택 사항에 대해 어떤 작업도 수행되지 않습니다.

   >[!CAUTION]
   >
   >첫 번째 일치 선택만 흐름 단계에 적용됩니다. 흐름 동작에서 &quot;선택 [추가&quot;의 순서를 변경하는 방법을 알아봅니다](reorder-add-choice-in-a-flow-step.md).

   >[!NOTE]
   >
   >**관련 문서**
