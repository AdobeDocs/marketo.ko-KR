# 데이터 값 변경 {#change-data-value}

## 개요 {#overview}

Marketo을 사용하여 필드의 값을 업데이트할 수 있습니다. 이 경우 다음을 사용합니다. **[!UICONTROL 데이터 값 변경]** 흐름 작업.

![](assets/image2014-9-22-11-3a15-3a34.png)

>[!NOTE]
>
>필드가 업데이트되지 않도록 차단할 수도 있습니다. 다음을 참조하십시오 [필드에 대한 업데이트 차단](/help/marketo/product-docs/administration/field-management/block-updates-to-a-field.md){target="_blank"} 을 참조하십시오.

## 사용량 {#usage}

1. 값을 변경할 필드를 찾아 선택합니다.

   ![](assets/image2014-9-22-11-3a18-3a29.png)

1. 원하는 값을 입력하면 완료됩니다!

   ![](assets/image2014-9-22-11-3a18-3a38.png)

   >[!NOTE]
   >
   >다음에서 토큰을 사용할 수도 있습니다. **[!UICONTROL 새 값]**.

   >[!TIP]
   >
   >에 &#39;NULL&#39;(따옴표 없음, 모두 대문자)을 입력할 수 있습니다. **[!UICONTROL 새 값]** 필드를 지웁니다. 다음을 참조하십시오 [필드 값 지우기](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/change-data-value/clear-field-values.md){target="_blank"} 을 참조하십시오.

   >[!NOTE]
   >

   >* [필드에 데이터 추가](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/append-data-to-a-field.md){target="_blank"}
