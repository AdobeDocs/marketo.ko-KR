# 스마트 캠페인에서 웹 후크 사용 {#use-a-webhook-in-a-smart-campaign}

웹 후크를 사용하려면 [](http://developers.marketo.com/documentation/webhooks/)흐름 동작으로 [고급 캠페인에](add-a-flow-step-to-a-smart-campaign.md) 추가합니다.

>[!NOTE]
>
>**가용성**
>
>모든 고객이 이 기능을 구입하지는 않았습니다. 자세한 내용은 영업 담당자에게 문의하십시오.

1. [스마트 캠페인](../../../../product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/create-a-new-smart-campaign.md)만들기

   >[!NOTE]
   >
   >Webhook은 트리거 캠페인에서만 사용할 수 있습니다.

1. 흐름 **탭으로** 가서 **호출 웹후크** 흐름 동작을 드래그합니다.

   ![](assets/image2014-9-22-15-3a8-3a2.png)

1. 웹후크를 선택합니다.

   ![](assets/image2014-9-22-15-3a8-3a5.png)

1. 스마트 목록에서 Webhook를 사용할 수도 있습니다.

   ![](assets/2017-05-02-10-54-38.png)

1. 마지막으로 흐름 단계에서 선택 **추가** 단계에서 Webhook를 사용할 수 있습니다.

   ![](assets/image2014-9-22-15-3a8-3a13.png)
