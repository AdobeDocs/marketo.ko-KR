# 스마트 캠페인에서 웹 후크 사용 {#use-a-webhook-in-a-smart-campaign}

를 사용하려면 [webhook](https://developers.marketo.com/documentation/webhooks/)에 추가합니다 [스마트 캠페인](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/add-a-flow-step-to-a-smart-campaign.md) 를 흐름 작업으로 사용합니다.

>[!AVAILABILITY]
>
>모든 고객이 이 기능을 구입한 것은 아닙니다. 자세한 내용은 영업 담당자에게 문의하십시오.

1. [스마트 캠페인 만들기](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/create-a-new-smart-campaign.md).

   >[!NOTE]
   >
   >Webhooks는 트리거 캠페인에서만 사용할 수 있습니다.

1. 로 이동합니다. **흐름** 탭을 선택하고 을(를) 드래그합니다. **Webhook 호출** 흐름 작업.

   ![](assets/image2014-9-22-15-3a8-3a2.png)

1. 웹 후크를 선택합니다.

   ![](assets/image2014-9-22-15-3a8-3a5.png)

1. 스마트 목록에서 Webhooks를 사용할 수도 있습니다.

   ![](assets/2017-05-02-10-54-38.png)

1. 마지막으로 Webhooks를 **선택 추가** 를 입력합니다.

   ![](assets/image2014-9-22-15-3a8-3a13.png)
