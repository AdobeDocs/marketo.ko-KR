# 대기 흐름 단계에서 특정 날짜 사용 {#use-a-specific-date-in-a-wait-flow-step}

를 사용할 수 있습니다 **대기** 흐름 단계를 사용하여 특정 날짜까지 스마트 캠페인을 통해 사용자의 여정을 일시 중지합니다.

1. 스마트 캠페인에서 **흐름** 탭에서 위로 드래그합니다. **대기** 흐름 단계.

   ![](assets/image2014-9-22-11-3a50-3a55.png)

1. 오른쪽에 있는 톱니바퀴 아이콘을 클릭합니다.

   ![](assets/image2014-9-22-11-3a50-3a59.png)

1. 에서 **유형** 드롭다운에서 을 선택합니다. **날짜**.

   ![](assets/image2014-9-22-11-3a51-3a27.png)

1. 다시 시작할 특정 날짜를 선택합니다.

   ![](assets/image2014-9-22-11-3a51-3a20.png)

1. 시간(선택 사항)을 지정하고 **저장**.

   ![](assets/image2014-9-22-11-3a51-3a13.png)
