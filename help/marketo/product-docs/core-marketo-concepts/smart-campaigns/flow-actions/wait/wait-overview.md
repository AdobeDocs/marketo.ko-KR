# 대기 {#wait}

## 개요 {#overview}

편리한 기능을 사용하여 스마트 캠페인 플로우에서 사람 일시 정지 **대기 단계**.

![](assets/wait-overview.png)

&quot;4시간&quot;과 같은 자연어를 어떻게 입력할 수 있는지 주목하세요. 작업 **not**&#x200B;를 입력합니다(즉, 4시간). 스마트 캠페인이 여전히 실행되지만 대기 단계는 무시됩니다.

>[!CAUTION]
>
>대기 단계의 기간을 변경해도 이미 해당 단계에 입력한 사람에게 영향을 주지 않습니다. 예: 5일 동안 대기 단계가 있고 사용자가 해당 단계를 입력한 다음 대기 단계를 7일로 변경합니다. 이 사용자는 다음 흐름 단계로 이동하기 전에 원래 5일만 대기합니다.

>[!TIP]
>
>대기 단계에 이미 있는 사람이 있고 대기 기간이 끝난 후에 해당 사용자가 진행되지 않도록 하려면 를 삽입합니다. [흐름에서 제거](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/remove-from-flow.md) 대기 단계 바로 뒤에 있습니다. 을 사용하여 제거할 사용자를 지정합니다. [선택 추가](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/use-add-choice-in-a-flow-step.md) 선택 사항입니다.
