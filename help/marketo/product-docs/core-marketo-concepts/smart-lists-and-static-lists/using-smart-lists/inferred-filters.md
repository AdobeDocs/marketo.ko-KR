# 유추 필터 {#inferred-filters}

누군가 웹 사이트를 방문하면 [Munchkin](../../../../product-docs/administration/additional-integrations/add-munchkin-tracking-code-to-your-website.md) 쿠키가 쿠키를 작성하고 시스템에 넣습니다. 우리는 그들의 IP를 특별한 데이터베이스에 올려놓고 모든 종류의 좋은 정보를 추론한다. 확인해 보십시오.

![](assets/image2015-4-27-13-3a36-3a9.png)

![](assets/image2015-4-27-13-3a30-3a48.png)

스마트 목록에서 이러한 필터를 사용하면 유추된 정보가 있는 사람이 결과를 얻게 됩니다.

>[!TIP]
>
>웹 활동 보고서에서 이러한 필터를 사용하십시오. 영업 담당자의 영역을 사용하고 지난 24시간 동안 웹 사이트 방문자가 포함된 맞춤형 일일 보고서에 가입합니다. 그들은 그것을 좋아할 것이다!
>
>* 방문한 웹 페이지 - 지난 24시간
>* 유추 상태가 해당 영역을 [선택합니다.]

>



이러한 익명의 방문자는 이메일 링크를 클릭하거나 양식을 작성할 때 자동으로 다른 사람으로 변환됩니다. 하지만 그들은 유추된 모든 정보를 유지한다.

>[!NOTE]
>
>**딥 다이브**
>
