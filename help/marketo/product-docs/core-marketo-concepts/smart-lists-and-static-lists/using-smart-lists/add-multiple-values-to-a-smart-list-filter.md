# 스마트 목록 필터에 여러 값 추가 {#add-multiple-values-to-a-smart-list-filter}

>[!PREREQUISITES]
>
>* [스마트 목록 만들기](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)
>* [스마트 목록 필터 찾기 및 추가](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/find-and-add-filters-to-a-smart-list.md)

캘리포니아에 있는 모든 사람을 찾고 싶지만 둘 다 저장하고 있을 수 있습니다 **캘리포니아** 및 **CA** 을 참조하십시오. 해당되는 모든 사람을 포함시키려면 두 가지를 사용할 수 있습니다 **시/도** 필터를 사용하지만 하나를 사용하면 더 쉽습니다.

1. 다음으로 이동 **마케팅 활동**.

   ![](assets/login-marketing-activities-2.png)

1. 스마트 목록을 찾아 선택하고 **스마트 목록** 탭.

   ![](assets/smarlist-choosefilters.png)

1. 다음을 클릭합니다. **+** 필터에서.

   ![](assets/smartlist-plussignhand-.png)

1. 왼쪽에서 값을 선택하거나 오른쪽에 값을 입력한 다음 **확인**.

   ![](assets/image2014-9-11-17-3a51-3a39.png)

빨리 해!
