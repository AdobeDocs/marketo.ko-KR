# 스마트 목록 만들기 {#create-a-smart-list}

스마트 목록을 사용하면 간단한 필터를 사용하여 특정 사용자 그룹을 찾을 수 있습니다. 만드는 방법은 다음과 같습니다.

1. 다음으로 이동 **마케팅 활동**.

   ![](assets/login-marketing-activities.png)

1. 스마트 목록을 만들 프로그램을 선택합니다.

   ![](assets/image2014-8-11-10-3a17-3a8.png)

1. 아래 **신규**, 클릭 **새 로컬 자산**.

   ![](assets/image2014-9-9-16-3a26-3a28.png)

1. 선택 **스마트 목록**.

   ![](assets/image2014-9-9-16-3a27-3a18.png)

1. 입력 **이름** 및 클릭 **만들기**.

   ![](assets/image2014-9-9-16-3a27-3a39.png)

   짜잔, 수고했어! 이제 몇 가지 필터를 추가하고 정의하겠습니다.
