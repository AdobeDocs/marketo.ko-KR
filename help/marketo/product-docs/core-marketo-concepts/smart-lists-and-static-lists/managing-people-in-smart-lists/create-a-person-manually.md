# 수동으로 개인 생성 {#create-a-person-manually}

Marketo에 사람을 끌어들이는 방법에는 여러 가지가 있습니다. 수동으로 만들어야 하는 경우 다음 방법을 설명합니다.

>[!CAUTION]
>
>Marketo의 기능 **not** 이모지가 포함된 이메일 주소를 지원합니다.

1. 로 이동합니다. **데이터베이스**.

   ![](assets/db-1.png)

1. 아래 **새로 만들기**&#x200B;를 클릭합니다. **새 사람**.

   ![](assets/two-2.png)

1. 개인 정보를 입력한 다음 **만들기**.

   ![](assets/three-2.png)

Marketo에서 한 사람을 빠르고 쉽게 추가할 수 있습니다!
