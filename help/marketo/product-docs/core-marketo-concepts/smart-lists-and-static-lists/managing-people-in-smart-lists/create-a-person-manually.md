# 수동으로 사용자 만들기 {#create-a-person-manually}

Marketo에 사용자를 초대하는 방법에는 여러 가지가 있습니다. 수동으로 만들어야 하는 경우 방법은 다음과 같습니다.

>[!CAUTION]
>
>Marketo은 **아님** 이모지가 포함된 이메일 주소를 지원합니다.

1. 로 이동 **데이터베이스**.

   ![](assets/db-1.png)

1. 아래 **신규**, 클릭 **새 사용자**.

   ![](assets/two-2.png)

1. 개인 정보를 입력한 다음 **만들기**.

   ![](assets/three-2.png)

Marketo에서 한 사람을 빠르고 쉽게 추가할 수 있는 방법입니다.
