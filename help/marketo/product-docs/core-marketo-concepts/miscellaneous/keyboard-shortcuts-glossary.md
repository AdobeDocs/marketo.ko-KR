# 키보드 단축키 용어집 {#keyboard-shortcuts-glossary}

다음 키보드 단축키를 사용하여 Marketo을 보다 빠르게 탐색할 수 있습니다.

## 영역 탐색 {#area-navigation}

| 키보드 단축키 | 영역 |
|---|---|
| Alt/Opt + A | Analytics |
| Alt/Opt + C | 캘린더 |
| Alt/Opt + D | Design Studio |
| Alt/Opt + F | 글로벌 검색 |
| Alt/Opt + L | 데이터베이스 |
| Alt/Opt + M | 마케팅 활동 |
| Opt + N | 관리 |
| Opt + O | 내 Marketo |

## 달력/예약 보기 탐색  {#calendar-schedule-view-navigation}

| 키보드 단축키 | 액션 |
|---|---|
| Alt/Opt + 위쪽 화살표 | 시간 내 뒤로 이동 |
| Alt/Opt + 아래쪽 화살표 | 시간 내에 앞으로 이동 |
| Alt/Opt + T | &#39;오늘&#39;로 이동 |

## 기타 {#miscellaneous}

| 키보드 단축키 | 액션 |
|---|---|
| ESC | 팝업 닫기 |
| 반환/입력 | 팝업 제출 |
| Ctrl + Alt/Opt + H | 양식 편집기에서 필드 숨기기 또는 숨기기 취소 |
