# 항목 세부 사항 보기 {#view-entry-details}

1. 달력에서 항목을 선택합니다.

   ![](assets/image2014-9-26-10-3a30-3a44.png)

1. 마케팅 캘린더에서 항목은 읽기 전용입니다. 편집할 프로그램으로 이동합니다.

   ![](assets/image2014-9-26-10-3a31-3a1.png)

>[!TIP]
>
>세부 정보 패널을 마우스 오른쪽 버튼으로 클릭하여 탐색 메뉴에 액세스하거나 편집기를 엽니다.
