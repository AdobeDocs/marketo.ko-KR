# 작업 영역별로 마케팅 캘린더 필터링 {#filtering-the-marketing-calendar-by-workspace}

마케팅 달력은 특정 작업 공간의 객체별로 필터링할 수 있습니다.

1. **[!UICONTROL Calendar]** 타일을 클릭합니다.

   ![](assets/2017-05-10-15-30-47-1.png)

1. 필터 패널에서 **[!UICONTROL Workspace]** 드롭다운을 선택합니다. 원하는 작업 영역을 선택합니다.

   ![](assets/image2014-9-24-11-3a34-3a6.png)

   잘됐네! 이제 해당 특정 작업공간에 빌드된 객체만 표시됩니다.
