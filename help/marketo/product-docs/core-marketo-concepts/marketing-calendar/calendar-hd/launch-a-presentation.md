# 프레젠테이션 시작 {#launch-a-presentation}

프레젠테이션의 보기 횟수 및 회전 빈도를 설정하고 나면 바로 시작할 시간입니다!

>[!AVAILABILITY]
>
>
>모든 고객이 이 기능을 구입한 것은 아닙니다. 자세한 내용은 영업 담당자에게 문의하십시오.

>[!PREREQUISITES]
>
>* [프레젠테이션 만들기](/help/marketo/product-docs/core-marketo-concepts/marketing-calendar/calendar-hd/create-a-presentation.md)
>* [프레젠테이션 사용자 지정](/help/marketo/product-docs/core-marketo-concepts/marketing-calendar/calendar-hd/customize-a-presentation.md)


>[!TIP]
>
>다음을 확인하십시오 **프레젠테이션 미리 보기** 시작하기 전에.

1. 클릭 **Launch**.

   ![](assets/image2015-3-20-14-3a4-3a18.png)

1. 클릭 **Launch** 한 번만 더 이렇게 하면 프레젠테이션이 안전하게 표시되도록 Marketo에서 로그아웃됩니다.

   ![](assets/image2015-3-20-14-3a5-3a34.png)

   >[!TIP]
   >
   >프레젠테이션이 새 탭에서 실행됩니다. 필요한 경우 외부 모니터나 TV로 탭을 이동하여 디스플레이할 수 있습니다. **전체 화면** 또는 **F11** Internet Explorer를 사용하는 경우 키보드에서
