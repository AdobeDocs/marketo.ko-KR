# 프레젠테이션 실행 {#launch-a-presentation}

프레젠테이션의 보기 횟수와 회전 빈도를 설정한 경우 이제 시작할 때입니다.

>[!NOTE]
>
>**가용성**
>
>모든 고객이 이 기능을 구입하지는 않았습니다. 자세한 내용은 영업 담당자에게 문의하십시오.

>[!PREREQUISITES]
>
>* [프레젠테이션 만들기](create-a-presentation.md)
>* [프레젠테이션 사용자 정의](customize-a-presentation.md)

>



>[!TIP]
>
>실행하기 전에 **프레젠테이션** 미리 보기를 확인하십시오.

1. 론치를 **클릭합니다**.

   ![](assets/image2015-3-20-14-3a4-3a18.png)

1. 론치 **를** 한 번 더 클릭합니다. 이렇게 하면 프레젠테이션을 안전하게 표시할 수 있도록 Marketing To에서 로그아웃됩니다.

   ![](assets/image2015-3-20-14-3a5-3a34.png)

   >[!TIP]
   >
   >프레젠테이션이 새 탭에서 실행됩니다. 필요한 경우 외부 모니터나 TV로 탭을 이동하여 **전체 화면** 을 클릭합니다. Internet Explorer를 사용하는 경우 키보드에서 **F11 **을 누릅니다.
