# 코드 조각 만들기 {#create-a-snippet}

스니펫은에서 동적 콘텐츠 블록으로 사용할 수 있습니다. **이메일** 및 **랜딩 페이지**.

1. 로 이동 **디자인 스튜디오.**

   ![](assets/designstudio.png)

1. 클릭 **신규** 그런 다음 **새 코드 조각**.

   ![](assets/image2014-9-16-8-50-4.png)

1. 필요한 세부 정보를 입력하고 을(를) 클릭합니다 **만들기**.

   ![](assets/image2014-9-16-8-3a50-3a14.png)

잘했어요! 다이내믹 컨텐츠에 대한 코드 조각을 만들어 작업을 간소화합니다. 이제 계속하셔도 되고 [새 코드 조각에 콘텐츠 추가](/help/marketo/product-docs/personalization/segmentation-and-snippets/snippets/add-content-to-a-snippet.md).
